【科技籽】Mac機有漏洞　木馬程式任裝？

Apple的系統在用家眼中，最可取的地方一向就其保安系統，電腦不容易中毒。進行電腦系統安全研究的Malwarebytes近日在一份報告中表示，他們在Apple的OS X 10.10系統發現了一個新的Zero-day漏洞，不法分子可以在這個漏洞中，不知不覺地植入木馬程式，令電腦中毒，威脅信息安全。
據有關安全人員指，他們測試OS X 10.10.4後發現一個新漏洞，而Apple方面還沒有發表有關評論。根據報告指，原來最新的OS X 10.10.5 Beta版當中，這個代碼為DYLD_PRINT_TO_FILE exploit的Zero-day漏洞仍然存在，幸好是Apple的升級系統OS X 10.11 El Capitan並沒有受到這個漏洞影響，相信Apple已經在升級版封閉了這個缺口。但如果你仍在用OS X 10.10.4或10.10.5 Beta，則千萬不要打開不明來歷的網站或應用程式。
