【科技籽】2,048階壓感平板　似畫真畫

【科技籽：出閘搜新】

寫意的寫生，不知大家幾耐冇試過？時代轉變，畫架與畫布，早已被插畫師們最愛的MacBook配繪圖畫板所取代；將兩者合一，就變成Wacom最新Cintiq Companion2，是Win8.1的平板，亦是2,048級壓感的畫布，今日找來新舊兩代插畫師測試兩代產物，最愛是誰？Companion2的功能與方便性無敵，但價錢亦是考慮點。

Win 8.1系統 可裝專業軟件

說到電腦用的繪圖筆，Wacom可說在設計師與畫師間的專業市場，手執牛耳！一部MacBook配上一塊Wacom繪圖畫板，是現下不少人的選擇。兩年前開始Wacom曾經推出平板式繪圖畫板，以最直觀的方法，在顯示屏幕之上直接揮毫，看似比「M+W」的組合更方便、更吸引！不過事實是仍未能大行其道，究其原因可能是系統問題？
以往這類平板繪圖板，大多以Android作系統，對一般用家的確是有大量免費軟件下載的優勢，但相對專業級用家，他們每每在一套專業軟件之上，浸淫十年八載的時間，所以今次全新的Cintiq Companion 2，首度以Win 8.1作系統，的確大大加強了它的實用性，須知大多專業級設計軟件，都會有PC及Mac版本，配合Cintiq Companion 2設有2,048階壓感和傾斜識別功能，再加跟機Pro Pen就能做出傳統筆刷、鋼筆或marker的畫風，用家終於可以完美地在 Adobe、Autodesk、Corel、Smith Micro、Pixologic等專業軟件之上，自由地做創作。

13吋aRGB芒 色域廣

全機焦點部份相信會是顯示部份，13吋寛角度顯示器，擁有2,560 x 1,440 QHD顯示能力及1,670萬色彩顯示，質素完全媲美13.3吋MacBook Pro的Retina顯示器，2,560 x 1600的解析度；不過更讓設計用家心動者，相信是Companion 2完全對應了aRGB的顯示制式，雖然市場上有好多更高解像能力，直達4K或5K的大吋數顯示屏幕，不過其中不少只能對應sRGB制式（只適用於網頁的RGB影像），看芒作畫時的確畫質細緻，但每當需要打印輸出時，完全就變成另一回事；Companion 2的顯示屏幕使用了Adobe倡導的aRGB，好處是色域更廣、更濃艷之餘，在顯示屏所見色彩，到直接列印為成品時，色彩會有完美的一致性，大大方便了專業設計用家的實際需要。
其他硬件配置都相當高，頂級PREMIUM版本使用了Intel Core i7處理器，配256GB SSD HDD、8GB RAM及Win 8.1 Pro，機價為$16,990。較平價的是STANDARD版本，使用了Intel Core i5處理器，配128GB SSD HDD、8GB RAM及Win 8.1，機價為$13,990。就現場實試，兩機在畫畫情況下相當順暢，不斷用兩指放大縮小，更換畫筆與顏色，都不會有任何遲緩感覺，如果說最實用的版本，個人認為會是STANDARD版，記憶體較少的問題，可以用外置式記憶體解決；不過如果你對3D繪圖有較大興趣，估計i7的PREMIUM版本才是你杯茶。

郭詠嘉：手眼一致 協調性高

說起郭詠嘉，大家可能仍停留在新聞主播的記憶，其實她自去年開始學習插畫，並於十一月舉辦首個個人畫展後，現職為自由插畫師，主攻水彩、人像及時裝插畫。全新的Cintiq Companion 2已試用了好一段時間：「我比較愛畫人像畫，所以人物的比例相當重要，以往的MacBook配繪圖板，手有手畫，眼要望屏幕，相對協調性比較低，新的平板繪圖板好處，是落筆的位置與線條，可以直接望着來畫，所以眼手的配合會更好，與一般以畫紙的手繪作畫，感覺非常相近。」

陳姵妏：完全毋須熟習時間

陳姵妏並不是甚麼繪畫名家，她只是一個普通如你我的多媒體設計與科技系1年級學生，平時她都有用MacBook配繪圖板的組合，在Coffee Shop的閒餘時間作畫:「香港的餐廳枱，一般都比較細，當放上了MacBook，好多時都會冇位置放畫板，就算枱面較大者，繪圖板都會凸出一半來，邊畫、邊搖晃，感覺相當差；今次是第一次試用平板繪圖板，感覺相當方便，好處是出外作畫時，可以攜帶更少更輕的繪圖工具；另外Companion 2的靈敏度相當高，筆觸的大細力度，都有相當高的如實反映，開初都會怕屏幕面會有跣筆的情況，實際用落，基本手感與一般繪圖板無異，而且一樣有自訂按鍵，在使用上完全毋須熟習時間。」

Gadget Guy評語

肥通：
高新科技是吸引，不過買機前要先冷靜；首先這是為專業人士設計的平板電腦，2,048階壓感都已屬繪圖畫板的專業級配置，但問題是你有否實際需要？肥通亦有不少攻讀設計系的朋友，窮苦學生多會用Surface Pro 2，10.2吋觸屏，配128GB SSD HDD及4GB RAM，現在剛下調的街價可以去到$5,980，只要加上一支Wacom繪圖筆，就有不俗的繪畫表現。
