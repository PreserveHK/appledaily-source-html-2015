醫述醫術︰乙型肝炎病毒 - 梁憲孫

54歲的王先生是乙肝病毒帶菌者，最近確診早期高毒性B細胞淋巴癌，需要接受化療及標靶藥物治療，估計有不錯治癒機會；抗癌治療抑制王先生的免疫系統，可令乙肝病毒轉趨活躍，誘發急性肝炎，甚至肝臟衰竭，可以致命。醫生化療前處方新一代抗乙肝病毒藥物，可抑制病毒，少有抗藥性，可有效預防急性肝炎。
全球有2億5千萬名乙型肝炎病毒帶菌者，每年因乙肝病毒感染而死亡的人數達50萬。全港有一成市民屬乙肝帶菌者，併發症包括慢性肝炎、肝硬化及肝癌。
香港的乙肝病毒帶菌者大多是出生後，與父母及親人親密接觸後受到傳染。初生嬰兒的免疫系統未成熟，未有能力清除病毒，感染後大多會成為乙肝帶菌者。血液中「乙肝病毒表面抗原」（Hepatitis B surface antigen, HBsAg）及「乙肝病毒中心抗體」（Hepatitis B core antibody, HBcAb）呈陽性反應，「乙肝病毒表面抗體」（Hepatitis B surface antibody, HBsAb）則一般是陰性。
孩子成長後，免疫系統較成熟，那時感染乙肝病毒，或有能力清除病毒，血液中「乙肝病毒表面抗原」呈陰性，「乙肝病毒中心抗體」（Hepatitis B core antibody, HBcAb）及「乙肝病毒表面抗體」則為陽性反應。
香港有一成人是乙肝病毒帶菌者，另有四成亦曾受感染，之後產生乙肝病毒表面抗體，對乙肝病有抵抗力。西方國家的情況稍有不同，外國的感染病例通常是成年人，大多感染後有乙肝病毒表面抗體，未有成為乙肝病毒帶菌者。
乙肝病毒疫苗有效預防感染，香港政府多年前已開始為全港初生嬰兒接種疫苗，防止嬰兒出生後從父母親感染乙肝病毒，成功令全港乙肝病毒帶菌者數目逐年下降，香港人出生的年輕一代極少是乙肝病毒帶菌者。
部份乙肝病毒帶菌者多年後會有併發症，包括急性和慢性肝炎、肝硬化及肝癌。近年多種有效對抗乙肝病毒藥物，可控制乙肝病毒，減少併發症機率，建議乙肝病毒帶菌者見醫生，商量應否服用抗乙肝病毒藥物。
