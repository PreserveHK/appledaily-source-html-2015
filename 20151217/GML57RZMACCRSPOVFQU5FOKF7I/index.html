【寵物食堂】寵物聖誕大餐 牛肉批配綠茶布丁

甚麼都愛放入口的貓貓狗狗，未必知道甚麼是聖誕大餐，但只要好味，牠們都無任歡迎！寵物籽找來李信瑩（Libby）教大家整愛心聖誕餐給愛寵品嚐。Libby見愛犬不是吃乾糧就是吃罐頭，感覺好dry，於是兩年前開始上網學整寵物小食，整小食前更會做足功課，確保選用對狗仔無害的食材。這次 Libby會教大家整「愛心牛肉批」和「雪地上的聖誕樹」（綠茶蛋糕布丁），主人不妨一試為愛寵下廚。

愛心牛肉批材料：

1. 免冶牛肉 50g
2. 雞蛋 1隻
3. 粟粉 10g
4. 燕麥片 15-20g
5. 紅蘿蔔 兩片
6. 忌廉芝士、水果 、西蘭花 適量

愛心牛肉批做法：

1. 將雞蛋、粟粉 、燕麥片和免治牛肉混合攪勻，盡量攪拌至所有栗粉都黏實其他材料
2. 攪勻後放入模具蒸20分鐘，脫模後便為牛肉批做裝飾
3. 在牛肉批上灑少量忌廉芝士，在忌廉芝士上放上少許西蘭花
4. 將兩片砌成心形的紅蘿蔔圍在忌廉芝士外面
5. 隨意在碟旁加上忌廉芝士和水果，Libby建議可用藍莓伴碟，對狗仔眼睛十分健康，裝飾後便完成

雪地上的聖誕樹分為乳酪布丁及綠茶蛋糕兩部份

乳酪布丁材料：

1. 純味乳酪 90g
2. 寵物奶 100g
3. 魚膠粉 6g
4. 水 20g
5. 角豆粉 適量
乳酪布丁做法：

1. 將魚膠粉和水混合攪勻，用熱水座溶
2. 把純味乳酪和寵物奶混合攪勻，並將攪勻的魚膠溶液倒入攪勻，直至魚膠溶液完全溶掉
3. 全部攪勻後倒入杯仔，放入冰箱冷藏3至4小時
4. 布丁成形後，在布丁面上灑上適量角豆粉便完成

綠茶蛋糕材料：

1. 低筋麵粉 35g
2. 綠茶粉 10g
3. 雞蛋兩隻
4. 水 15g
5. 綠茶忌廉芝士、玉桂粉 適量

綠茶蛋糕做

1. 將蛋白和蛋黃隔開，蛋黃倒入水中
2. 先將蛋白打至起泡
3. 打好蛋白後，倒入綠茶粉、低筋麵粉、兩隻蛋黃和水，全部攪勻
4. 攪勻後倒入聖誕樹模具，放入焗爐以160度焗25分鐘
5. 焗好後，把綠茶蛋糕放在布丁上，再以綠茶忌廉芝士圍邊，做出聖誕樹效果
6. 加完綠茶忌廉芝士，灑上少許玉桂粉便完成
