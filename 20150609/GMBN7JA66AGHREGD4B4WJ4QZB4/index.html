【汽車籽】新世代Civic　未出先破紀錄

Type R迷實在餓很久了，由本田宣佈開發新世代Civic Type R，到在年初日內瓦車展首度曝光，一直都是車迷焦點。日前廠方終於釋出新車詳細性能規格，讓廣大車迷心裏有個底。首先跟隨渦輪增壓大潮流，歷史性換裝Turbo引擎，1,996cc直四i-VTEC能擠出驚人的310匹馬力，配合六速手動波箱，剛以7分50秒63刷新了紐堡靈北賽道最新圈速，成為最快量產頭轆車款，可謂繼四驅的戰神GT-R後，又一令歐系跑車頭痛的一員。

新Civic Type R將在第二季先行於歐洲發售，其他地區預計今年年尾或2016年登場，至於香港，問過代理合群汽車，在今年年尾引進香港是「樂觀」的。縱觀性能及全車落重料設備，估計車價要去到「四字尾」，不過新Civic Type R勢成全球車迷搶購目標，有錢都未必有貨，一眾Type R迷要儲定彈藥了。

記者：徐影龍
