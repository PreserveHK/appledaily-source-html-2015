洪秀柱被指碩士學歷造假

國民黨總統參選人洪秀柱，被指學歷造假。名嘴周玉蔻在政論節目中指出，洪在美國密蘇里州立東北大學（Northeast Missouri State University）修讀的教育碩士，只是「暑期專修班」，學歷連台灣的教育部都不認可，質疑她因為學歷經不起考驗而不敢訪問美國。

反駁「學歷不等於能力」

周玉蔻在三立電視的政論節目《新台灣加油》中指出，洪秀柱修讀的碩士課程，每年上課6星期，連續3年即可取得證書，嘲諷「這是一個6個月的教育碩士」，她的學位是「假貨」；她又稱密蘇里州立東北大學在1996年改名為杜魯門大學（Truman State University），東北大學變成幽靈大學，稱洪將「偽造文書」刊登在選舉公報應該退選。
面對周玉蔻的指控，洪秀柱昨日出示她取得的證書，表示她利用寒暑假攻讀學分班，雖然不符合教育部相關時數、停留時間規範，「但學習過程，與學分的取得，還有證書都是真的」。她又指修讀碩士是秉持終身學習精神，並非想藉此讓教職再升等或在職場加分，她強調「學歷不等於能力」，批評高學歷的人不一定具備能力。
台灣教育部高教司長黃雯玲昨稱，即使洪的學位是暑期班或是遠距教學，只要學校認證的證書非偽造，學歷就有效。中選會也說，洪刊登在立委選舉公報的學歷沒問題。
台灣《蘋果日報》
