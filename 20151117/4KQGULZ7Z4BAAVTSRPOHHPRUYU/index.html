【寵愛生命】喵星人必備貓砂　用法分享

貓砂是養貓的必需品，除非你的貓曉得上人類廁所，否則那小片砂地就是牠們解決大小便的地方！神奇地，喵星人們都會自動懂得在砂上掘出小洞如厠，方便後又會企理地將洞填好，避免臭氣四散。

我也是最近看New York Times的網站，才得知原來貓砂是美國人Edward Lowe於1947年時發明。他是一位退伍軍人，在父親開設的工業用土壤公司上班。某寒冬，和鄰居閒聊間得知他的貓平日都在沙泥上如厠，泥土結冰後，人貓都煩惱。於是，熟悉各式泥土特性的Edward就為鄰居調配出一種特別的貓砂，成功解決問題後，Edward靈機一動再做了10袋5磅裝的貓砂，並在寵物店內寄賣，售價6毫，卻乏人問津。Edward索性讓店鋪老闆將貓砂送出，很快就得多位貓主的喜愛，訂購貓砂。Edward從此穿梭不同的貓展，為每個攤位提供貓砂，以換取位置寄賣自家的出品，貓砂自此被追捧，廣泛使用。

隨時代發展，Edward所發明，那種不凝結的貓砂(Non-Clumping Clay Cat Litter)漸被淘汰，取而代之是後來一位科學家William Mallow所研製，遇液體便會凝結成一團，便於剷走的貓砂。

這10餘年間在一片環保聲中，貓砂再有新發展。品牌推出以再造紙、以松木、粟米、黃豆和咖啡豆殼等作材料的貓砂，甚至有用矽膠製成的水晶珠砂、可沖進人類厠所的可分解的貓砂。

貓砂的種類：
(1) 不凝結泥土貓砂(Non-Clumping Clay Cat Litter)
優點：便宜又經濟
缺點：由於不凝結，因此清理較麻煩。

(2) 凝結泥土貓砂(Clumping Clay Cat Litter)
優點：尿液會凝固成團狀、方便清理。貓抓起來沒有粉塵，不怕吸入有害物質。
缺點：小貓或會偷吃，吃下肚會做成腸道不適，甚至阻塞，嚴重甚至會致死。

(3) 矽膠水晶貓砂(Silica-Gel Cat Litter)
優點：矽膠中含二氧化矽，跟日常所用的乾燥劑類似。便宜經濟、吸水力強、吸味能力亦佳。
缺點：圓形顆粒容易被貓帶出沙盆，不規則形狀的顆粒不適合小貓及患有關節炎的貓。

(4) 紙貓砂(Paper Cat Litter)
優點：以循環再造紙生產，1975年由日本的Paperlet公司生產。分為凝結型和不凝結型，可隨廁所水排走，能作燃燒處理。
缺點：吸味能力一般。

(5) 木削砂(Wood Base Cat Litter)
優點：以木材做成，因此帶天然木香，亦能分解。
缺點：木削可能會飛散。

(6) 豆腐貓砂(DouFu Cat Litter)
優點：以豆腐渣廢物做成，多數會凝結，方便清潔，亦可隨廁所水排走。
缺點：不太像泥沙的質感，有些貓未必喜歡用。

(7) 其他植物類貓砂(Plant Base Cat Litter)
優點：以咖啡豆殼、粟米、小麥等物料製成，很天然。
缺點：跟豆腐貓砂一樣，因為不太像泥沙的質感，有些貓未必喜歡用。

貓砂使用時需注意的：
(1) 貓砂假如形狀過於粗大，對小貓和患有關節炎的貓都不太好。
(2) 貓砂至少需2吋深，因為有些貓咪很喜歡掘砂，甚至會掘到盆底，如果貓砂太淺，貓的排泄物就會積在盆底。
(3) 凝結貓砂每次清理時只需將凝結成塊狀的貓砂清走，未污染的貓砂可繼續用，不時添上新的貓砂補充即可，不過亦需定時整盆換掉，及保持沙盆清潔。
(4) 貓廁所若有異味，可以加些食用蘇打粉在貓砂內代替用化學的除臭劑。
(5) 留意貓的排泄物，有助判斷愛貓的健康情況。
