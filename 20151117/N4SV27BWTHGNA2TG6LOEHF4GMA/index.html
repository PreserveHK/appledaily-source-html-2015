巴黎因果 - 鄧達智

說起少女自殺事件，二人未過19歲，本來純情不過少年同學初戀，少男透露需全心全意投入功課，少女當作世界末日，那天上門探訪，二話不說，從少男家33樓露台縱身跳下，獨女，父母如何浮過餘下生命海？
汪汪聽了說：因果！
她前生作的孽因，今生必須一跳，那是果，生她養她的父母命帶相關，亦然。
如何度過？
沈澱面對，所有質疑、憤怒、悲愴、甚至跳埋一份也無濟於事。
唯一處理：咬緊牙關處理後事，心情收復把前因後果淡化處理。
命中注定，她必須一跳！
發生了，除卻淡化面對，別無他法。不斷問為什麼？只會長浸未死已入鬼門關。
留在世上的人縱使哭斷腸，那果熟透了，掉下來了，能做的？
將它掃乾淨。
巴黎，只是一個開始，類同慘劇成了西方社會無可避免課題。聖誕節快來了，火樹銀花，Jingle Bells，Jingle Bells，Jingle all the way，ISIS Claus is coming to town！小心興致勃勃節前節後購物，尤其中國人，過去幾年倫敦Selfridges Boxing Day大減價，徹夜排隊輪候，猶如貧窮幾世未曾購買過，擁物得慰藉的中國人都是首先連奔帶跑拿個第一，甚至第二、三名衝入場。今年小心，人眾的地方少點去，真的，買少一兩個 Gucci、Dior手袋甚至洗面Mask、衛生巾……不會死。
西方社會能做些什麼？
軟方面積極：以外交途徑尋找缺口談判？ISIS並非國家，你去那裏找他們的公關？提供幕後支援必有巨掌，英美那麼多007，與其周旋，甚至殲滅？不過，巨掌何止一把？甚至007們頂頭上司。
硬碰應戰：你去那裏跟他們打仗？無邊界的國度處境極高明，你們炸，死的只是中東地區連年戰亂的無辜蟻民，繼而製造更多難民。聖戰鬥士可能就在巴黎第六區你樓下的14歲鄰居，又或者倫敦不一定南面，而是W2每日乖乖上學名校16歲少女，你如何搜出緝捕？
小偷都未能捉到過，就是捉到，九萬個原因兩小時內，捉蟲入股人道理由教他們放人，不少歐洲國家法國尤甚，你以為他們容易執行？
Facebook以百萬人計將面孔加上紅、白、藍之餘，有人將中東難民群照鑲在小布殊年代正副美國兩總統一塊，上款：The True Evil: Greed（真正魔鬼：貪婪）。中東難民照片印上：They Didn't Create ISIS（他們沒有製造伊斯蘭國）。布殊正副兩總統：They Did（伊斯蘭國由他們製造）。
二戰之後，西方殖民侵略陣營為勢所迫離開中東、非洲，為石油、為戰略形勢、為貪婪，都未曾真正離開過。因由自生，果報眼前。
