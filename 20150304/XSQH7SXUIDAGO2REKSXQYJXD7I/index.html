港情趣：手錶Memo布袋　鋁罐盡其用

驟眼看，確實很難發現這些動物模型是用汽水罐做成，一家本地設計公司運用鋁罐罐底製成手錶Can Watch，再利用剩餘的鋁罐罐身做成動物模型便條紙Can Create Forest，實行物盡其用。負責人Kat Ling開宗明義說其設計宗旨是「Turn Neglect into Shine」，「不論是物料還是和我們合作的夥伴，希望將人們忽視的東西，透過設計將其發光發亮。」
記者：劉東佩
攝影：譚建章

Kat說Can Create Forest的誕生，是想解決鋁罐罐身的問題。他們設計的手錶以鋁罐罐底來做錶殼，每個月固定生產一定數目，這同時卻也意味着剩下了很多鋁罐罐身，「我們時常思考如何將汽水罐物盡其用，尤其是罐身能否應用在不同的產品中。」見到坊間有越來越多人用軟木來做布袋，他們靈機一觸，將軟木布和鋁金屬兩種輕質物料結合在一起，設計了一系列產品，軟木布手袋Can Carry、證件套Can Carry Pass、筆袋Can Carry Pen以及可愛的Can Create Forest動物模型。「汽水罐罐身很光滑，用油性筆畫完可以拭擦，心想這不就可以當成memo紙嗎？於是將罐身及軟木布結合，做成有memo紙功能的動物模型。」

鋁罐升級做手錶

Kat曾在理工大學修讀產品設計，畢業後做過與設計相關的工作，有感於工作沉悶，遂與兩位同學Kenneth Ho與Samuel Siu成立Alchemist Creations，利用Upcycling（升級再造）的方法做設計。經過一番研究與討論，他們決定用鋁罐實踐自己的想法，「一來鋁罐容易收集，二來鋁罐罐底的設計也很漂亮，可以利用其已有設計再改造。」
他們向工場附近的餐廳收集被丟棄的鋁罐，經由清洗和挑選後，將鋁罐底部進行打磨、拋光等處理，加工成為錶殼與錶面，再加入手錶零件和錶帶，成為環保手錶Can Watch。Kat說當初也不懂得手錶設計，幸得錶行老闆免費做顧問，逐漸完善設計。手錶在二○一一年香港Detour展覽公佈時已有不少迴響，後來參加日本東京設計師周也有不錯反應，令他們相信環保設計其實有得做。之後他們還利用鋁罐做了些新嘗試，設計了枱燈及鋁罐袋。

由自閉人士生產

由設計到生產，整個過程都是在香港本土完成，當初他們的工場設在灣仔聖雅各福群會，錶帶部份則在將軍澳基督教家庭服務中心生產，將中心一些被棄置的皮革變為錶帶，後來乾脆把整個工場搬到將軍澳，簡化操作的步驟，也令生產過程更順利。現在Can Watch的切割、打磨及錶帶製作都由基督教家庭服務中心的員工負責，這些員工雖有自閉症等障礙，但在導師的指導下均能完成工作。公司也會因應工場員工的長處及技能，再想在設計上可以如何配合，與傳統設計師理念先行的想法不同。Kat希望透過產品帶出背後的信息，以設計去關注一些社會議題，並傳達他們的價值觀。

得紅點設計獎　在港無人關注

二○一三年初，他們膽粗粗將Can Watch申請了德國紅點設計大獎（Red Dot Design Award），「申請時手錶還有些小瑕疵，不過紅點很欣賞我們由物料到生產過程背後所帶出的訊息。」Alchemist Creations雖得到世界頂級設計大獎的肯定，不過在香港卻沒引起太多關注，筆者也是今年初在香港知專設計學院看紅點設計大獎展覽時，才赫然發現本土設計及製作的Can Watch名列其中。Kat坦言香港人對本土創意產業認識還不夠，「同一樣的價錢，大家會首先考慮著名品牌，而未必是香港品牌，我們也不能只依賴香港，畢竟市場較小。」她說得到紅點設計大獎後，要進入外國市場也會較容易。問及公司的將來，Kat說會繼續圍繞鋁罐設計不同產品，利用自身的設計及工場的設備去做不同的嘗試。月前他們推出了小尺寸的Can Watch，現階段也正在研究生產Can Watch機械錶，希望將公司的設計提升到另一個層次。
