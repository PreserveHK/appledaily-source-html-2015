【汽車籽】壓軸登場　黑玻璃超豪跑車

【汽車籽：貴駕光臨】
臨近年尾，以為再沒新車發佈，誰知平治依然賣力，於澳門發佈全新C-Class跑車，還要是雙門Coupe版本，實在讓人萬分驚訝。雖然要待明年才有機會一試，可是眼前這台AMG C 63S Coupe，既是最高級的Edition 1，更是量產前的pre-production版本，全球只得兩台，好些設計不會加入行貨，當然要把握機會睇真啲！

平治近年的跑車設計越來越有個性，手工亦不斷精益求精，初遇這台AMG C 63S Coupe，車身拉花及黑白色調已先聲奪人，並悄悄告訴大家它是Edition 1版本，可以預期內櫳用料有多豪華。車頭用上招牌鑽石形鬼面罩，下方的進氣位不但尺碼巨大，還加入碳纖物料來豐富細節，更具個性，頭唇及裙腳均塗上黑色，加強整體跑感。其流線型車身亦很有心思，車頭有稜有角、霸氣逼人，典型雙門跑車的細長腰線，加上圓滑車尾的弧形設計，令人一見難忘。車尾還有一個「亮點」，就是沒有打開尾箱的設置，把手、按鈕統統欠奉，換言之，只能依靠車匙或車內按鈕來打開尾箱。

擺脫限制　設計刁鑽

據了解，當日發佈的是量產前的版本，即pre-production版本，廠方可以不用理會各個地區的法例限制而製造出最完美的出品，全球只有兩台。因此，這台AMG C 63S Coupe才會用上香港行貨不會引入的黑色玻璃，平添一份神秘感，亦讓跑感更為突出，是普通玻璃無法造到的效果。
由於它是Edition 1版本，內櫳豪華自是情理之中，不但採用大量皮革及麖皮物料，為桶椅及軚盤帶來高級舒適手感，並且用上黃色縫線而非傳統紅線，亦是平治Edition 1與眾不同之處。中控台雖是大路設計，可是廠方特意增大整體空間，更加入名牌IWC時鐘來畫龍點睛。尋寶之旅還沒結束，在乘客座位前方的儲物箱，竟然藏有一個香薰座，讓噴出的冷氣帶有淡淡清新幽香，玩到這般刁鑽，目的只是讓全車的豪華感再度昇華。硬要挑剔，電動桶椅的移動速度實在太慢，而且移前的空間亦都不夠寬闊，出入後排座位實在是件苦差。雖說雙門跑車的後座多為應急之用，很少人願意坐，但如此設計實在太過趕絕外母了吧，希望明年到港的版本能夠改善一下。
性能上，因為明年才能試車，暫時未知它有多威猛，若從數字上看，4,000cc V8 BiTurbo引擎、馬力輸出超過500匹，低至1,750轉便可以發揮700Nm峯值扭力，真是聽到都興奮。我有幸一撻引擎，隆隆的引擎聲和排氣聲足以讓我熱血沸騰，真期待試車的一刻！

記者點評

讚︰車身設計兼顧跑感及豪華感，車櫳空間十足亦很舒適。
彈︰電動桶椅的移動速度甚慢，前移空間亦不足讓乘客走進後排。
