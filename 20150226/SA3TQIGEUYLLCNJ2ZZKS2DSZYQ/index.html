KARA落選成員遭解約So Jin疑抑鬱跳樓亡

KARA落選成員遭解約
So Jin疑抑鬱跳樓亡

韓國女團KARA去年大地震，Nicole及姜知英先後離隊，KARA其後透過節目《Kara Project》從七名練習生中挑選新成員，當中許英智勝出。不過，昨日韓國方面就驚爆當時未能加入KARA的練習生So Jin，於前日在其住所跳樓自殺，終年二十三歲。據知她在一個月前遭公司解約，明星夢碎，加上又有抑鬱症，可能因此尋死。
撰文：金美兒

韓國傳媒昨日驚爆，於去年六月以練習生身分參加選秀節目《Kara Project》希望加入組合KARA，但最終落選、年僅二十三歲的So Jin，於前日韓國時間下午約二時七分，於其居住的韓國大田住所十樓跳下，管理員其後在花槽中發現她，立即報警將So Jin緊急送到醫院，可惜So Jin最終搶救無效身亡。警方表示So Jin生前患有抑鬱症，雖現場未發現她留下任何遺書，初步估計她是跳樓自殺。

好像受很多傷害

So Jin的所屬公司DSP Media昨日公開So Jin的死訊，並表示：「昨晚（前晚）收到So Jin家人的來電，對於長時間一直以練習生身份活動的So Jin突如其來的消息感到非常心痛，家屬們現正處於傷心時期，希望大家可以刪除有關故人的新聞，讓家屬安靜地好好度過最後的時間。」
DSP Media方面強調至今仍未找到So Jin的遺書，不能確定So Jin去世的原因，所以希望傳媒不要揣測So Jin的死因，誇大她自殺的新聞。DSP Media亦表示已於一個月前與So Jin解約，她已不再是旗下的練習生，亦不是準備出道的新女團April的成員。有估計So Jin明星夢碎，可能因此而承受極大壓力，最終踏上不歸路。
So Jin去世消息傳出後，粉絲們都紛紛在網上留言表示難過及惋惜，而女歌手嘉熙亦於其twitter內傷心留言：「令人可惜的So Jin，明明不像是這樣的，一定不像是這樣的。」而So Jin生前在參演《KARA Project》時，曾在節目上表示：「雖然好像會受到很多傷害，但我會努力練習直到能夠在舞台上好好表現為止。」更指座右銘是享受生活。

另組女團不成功

So Jin在《KARA Project》憑着活力又性感的演出得到粉絲的支持，更成功躋身KARA候選成員最後四強，可惜，So Jin最後落敗給較她年輕兩年、現已加入KARA成為隊員的許英智。
節目完結後，粉絲們都紛紛希望So Jin跟其他落選的練習生可以另組一個女團，可惜一直未有So Jin出道的消息。

Amber男仔頭打扮受委屈

So Jin疑似因未能出道做歌手而自殺，而作為女團f(x)成員的Amber則在早前演出節目《4種秀》時，自爆曾因中性的打扮受盡委屈，想過自殺。Amber透露無論在韓國或其他國家，因為自己打扮中性而備受嘲諷、欺負，甚至孤立。Amber說：「常常感到很寂寞，因為得我一個人，沒有人理解我要走的路，有想過自己到底哪裏做錯了，亦有過想自殺的念頭。」
