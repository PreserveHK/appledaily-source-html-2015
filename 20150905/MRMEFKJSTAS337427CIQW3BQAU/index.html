單挑內地網民句句有火
𡃁妹撐港足微博被河蟹

內地上河蟹事件時有發生，「中港大戰」前晚上演，香港足球隊成功抵擋中國隊進攻，逼和中國隊0:0。昨日李綺雯（𡃁妹）為撐香港足球隊在微博貼文又出事！
多年前曾與香港足球隊成員陳肇麒（陳七）拍拖、熱愛看足球運動的𡃁妹亦十分關注前晚的「中港大戰」，分別於微博和Instagram連發兩段貼文力撐港隊逼和中國隊「好勁揪」。但其中一段轉貼鄭智鬧葉鴻輝是「狗」之報道，hashtag住：「那位 # 中港大戰 # 進不了球的 # 中國足球隊隊長 # 鄭智 # 賽後罵 # 香港守門員 # 葉鴻輝 # 是狗」再配上滴汗和兩個代表鄙視的emoji。惹來大批內地網民如螞蟻一樣纏上來，左一句「港獨」右一句「人點同港畜比呀」、「都是中國人不是嗎？」不堪入目。
有內地網民指𡃁妹挑撥離間，李綺雯回應說：「發生了就是事實，沒有挑撥離間。」也有網民指類似事件已非新鮮事，𡃁妹亦回覆：「不新鮮早有前科。」

秋生揶揄中國隊輸品

不過，貼文之後在微博離奇地消失，疑遭網站河蟹。𡃁妹昨日在電話中表示不知情：「我都唔知點解喎？」是否自己刪除？她說：「唔係呀！」是否被消失？她說：「係呀。」但Ig貼文仍在，只是微博被刪走，會否感到無奈？她說：「我都唔知幾時冇咗，應該係有用戶向新浪投訴。」

黃秋生前晚亦有在facebook留言：「如果呀鐵龍輝係狗，咁你連隻狗守龍門都射唔入，哎……真係唔X知你係乜嘞？（乜嘢省隊，縣隊？乜都唔係，係小島隊OK？小島隊同國隊打和OK？）大笨象同隻烏蠅打和，大笨象輸埋個品。」昨日秋生在電話補充說：「原本都驚驚哋（輸波），但港隊今次真係臨危不亂，打出team work。」
