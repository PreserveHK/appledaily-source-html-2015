C AllStar騷晚晚超時罰得爽

C AllStar一連三場「C AllLive演唱會2015」前晚圓滿結束，四子忘形高歌，三晚都超時完騷，Jase除浴袍晒胸肌，跳辣身舞時被女舞蹈員胸壓；King和嘉賓顏卓靈合唱，大跳Tango；King邀請女粉絲上台重演MV，女粉絲將面貼King的胸前，On仔叫女粉絲替King做人工呼吸，最後釗峰在演唱會感動到喊。此外，還邀請了大前輩蘇永康做嘉賓。
C AllStar每晚都超時被罰錢，前晚12點完騷，不過他們好開心說：「唔知罰幾多，叫霞姨畀少兩個飯盒，其實我哋係罰得好爽，我哋未試過罰錢，咪畀佢罰吓，又唔係冇錢畀佢罰，𠵱家觀眾話好抵睇，480蚊可以睇到4個鐘。」栢芝未有前來撐場，釗峰說：「唔緊要，佢係其中一個，相信佢見到我哋good show都好開心。」
