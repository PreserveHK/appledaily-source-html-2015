華麗重賞Balmain X H＆M

【紐約直擊】H&M跟一線大牌聯乘，例牌引起哄動。今年5月宣佈與法國品牌Balmain合作後，千呼萬喚的這個系列於香港時間前日凌晨在紐約舉行時裝騷，即使在fast fashion模式下，Balmain的couture式重手工仍統統到位，挾着華麗氣勢隆重登場。
採訪、攝影：趙潔恆

兩大品牌H&M、Balmain攜手打造的聯乘系列Balmain X H&M 注定成為話題，前日凌晨在紐約舉行的時裝騷，不但吸引Balmain友好的女模兵團Gigi Hadid、Kendall Jenner、Jourdan Dunn、Coco Rocha及露絲亨廷頓韋莉（Rosie Huntington-Whiteley）到場，還有英國It girl Alexa Chung、女星戴安歌露嘉（Diane Kruger）做front row嘉賓，人氣賺盡。

跳舞開場

甫開騷一大班舞蹈員表演節拍強勁的Vogueing舞蹈，在樓梯與天橋間舞動起的士高般的熱鬧氣氛。然後模特兒魚貫出場，示範以Balmain Nation為題的超華麗設計。
女裝剪裁盡見Balmain的DNA，拿手墊膊輪廓、貼身body-con dress、性感深V線條，而用料與手工也見高級訂製的影子，巴洛克提花天鵝絨、釘滿25,000顆珠飾連身裙、金色刺繡與綁繩細節軍裝、極鮮艷的碧綠與桃紅閃石、閃片裙子，重手工與高度bling bling的設計充滿舞台王者風範。

男裝呼應

同時展出的男裝則包含sporty、街頭時尚元素，除了型格的長身軍褸、拼色biker皮褸外，金線刺繡徽章的外套與黑色tuxedo等具代表性的glam look也與女裝呼應，建構平易近人版的Balmain之作，系列將於下月5日全球公開發售。

Olivier Rousteing的Baby

H&M今次與年僅30歲、任Balmain創作總監四年多的Olivier Rousteing合作也是走新世代的步調。Olivier與H&M創作顧問Ann Sofie Johansson在騷前的記者會上，分享創作點滴。Olivier表示系列由他在Balmain幾年來的部份設計為藍本：「我想做到訂製服的效果，但知道要控制成本，有些技術要棄用，又要在限制下做到最好。今次無論布料、手工都做出訂製感覺。」
提到系列中Olivier最喜愛的作品，「那件釘上珍珠的孖襟外套，令我想起在Balmain的第一季。我當時就像baby，幾年間成長轉變很多。」曾為Roberto Cavalli工作的他被問到有否就這次合作，與其他曾與H&M聯乘的設計師傾談，他就一展自我個性：「沒有亦沒需要。」

BSB出場回到90's

好戲在後頭，時裝騷謝幕後，眾模特兒未有回到後台，反而齊齊聚到台前，待司儀宣告神秘嘉賓、90年代當紅男團Backstreet Boys出場，全場高漲情緒再推高，時裝天橋變成流行曲舞台。
BSB穿上Balmain x H&M男裝，獻唱經典金曲《I Want It That Way》、《Larger Than Life》，勁歌熱舞全場呼歡，Olivier Rousteing等在台下都勁high跟住唱，個個回到小孩時代。
