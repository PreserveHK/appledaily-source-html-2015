【科技籽】唔用MacBook有乜揀？　三星跑分最勁

【科技籽：GG鬥室】
新一代MacBook推出後，不少人對它又愛又恨，愛它sharp得嚇人的Retina屏幕、不足一公斤的輕巧機身，以及九小時續航力；恨它只有一個USB-C端子，出外公幹還得多帶一堆轉接線，而且不能一邊充電一邊使用USB裝置。其實，除了MacBook，市面還有哪些配備同等處理器、方便文書工作的手提電腦？為此，Gadget Guy精心挑出三款筆電，跟MacBook埋「機」互撼，分個高下，當中Samsung ATIV Book 9成功勇奪「跑分王」的稱號。

慣用電腦處理文書工作的人，大多選擇Windows系統，並非說Mac OS不中用，只是若在其他機型開啟Office檔案，或有出錯等問題，讓人不得不續留Windows懷抱。而新一代MacBook用上Intel Core M處理器，效能介乎i3與i5之間，雖然符合長時間處理文書工作等要求，但只設一個端子應付充電及擴充功能，讓它予人實驗品的感覺，無論其他條件如何吸引，還是讓人卻步。

Windows機勝在夠方便

相對地，Windows機的產品周期較短，設計亦沒有那麼完善，若與MacBook相比，多少也有小瑕疵。但以應用層面來說，則完全沒有問題。因為當MacBook用家以單反相機拍照後，想把檔案抄入電腦，先要找來讀卡器，然後利用轉接線才能插進全機僅得一個的USB-C端子，但Windows用家早已藉內置讀卡器輕鬆辦妥。所以，Windows機仍有其賣點，我們找來三部同樣配備Intel Core M處理器的手提電腦來作比較，分別是Samsung ATIV Book 9、Asus Zenbook UX305及Lenovo Yoga 3 Pro，四者雖以處理文書工作為主，但定位、定價卻大不相同，一於看看它們的實際表現到底如何。
我們安排了三項測試：先用跨平台軟件Geekbench測試電腦的綜合效能（主要是中央處理器），數字越大，效能越佳；其次是從SD卡複製一個2.1GB（2,150MB）檔案到硬盤，看看抄寫速度；最後則挑戰文書工作以外的能力，利用剪接軟件Adobe Premiere Pro輸出一段影片（片長2分11秒），測試各機的影片編碼速度。

Apple MacBook　鍵盤稍硬

屏幕：12＂ （2304×1440）
處理器：Intel Core M 1.10GHz
記憶體：8GB
硬盤：256GB SSD
尺寸：0.52＂（H）×11.04＂（W）×7.74＂（D）
重量：0.92kg
續航力：約9小時
端子：USB-C×1，3.5mm耳機
售價：$9,998起
http://www.apple.com/hk

評價：新的機械式鍵盤極薄，但有點硬，打字需要一點力度，持續使用手部容易疲倦。若能夠接受這一點，以及只有一個USB-C端子，即使用Boot Camp來安裝Windows，亦是個好選擇。

Asus Zenbook UX305　靚芒抵玩

屏幕：13.3＂ （1920×1080）
處理器：Intel Core M 800MHz
記憶體：8GB
硬盤：256GB SSD
尺寸：0.5＂（H）×12.8＂（W）×8.9＂（D）
重量：1.18kg
續航力：約7小時
端子：USB 3.0×3，micro HDMI， SD讀卡器，3.5mm耳機/米高風
售價：$7,998
http://www.asus.com/hk

評價：它的假想敵明顯是MacBook Air而非MacBook，13＂ 1080P屏幕用來看影片也綽綽有餘，加上多個輸入端子，實際方便。可惜機身稍大，續航力稍遜，但考慮到價錢因素，也算不錯。

Lenovo Yoga 3 Pro　觸控反芒

屏幕：13.3＂ （3200x1800）觸控
處理器：Intel Core M 1.30GHz
記憶體：8GB
硬盤：256GB SSD
尺寸：0.5＂（H）×13＂（W）×9＂（D）
重量：1.2kg
續航力：約7小時
端子：USB 3.0×2，USB 2.0×1，micro HDMI，SD讀卡器，3.5mm耳機/米高風
售價：$14,198
http://www.lenovo.com/hk

評價：本次測試的電腦之中，唯一具觸控功能，亦是最早使用Core M的一款。最大特點是屏幕能360°反轉，化身成平板電腦，用來做presentatiion相當方便。可惜定價高，實在找不到購入理由。

Samsung ATIV Book 9　擴充性佳

屏幕：12.2＂ （2560×1600）
處理器：Intel Core M 1.10GHz
記憶體：4GB
硬盤：128GB SSD
尺寸：0.46＂（H）×11.18＂（W）×8.38＂（D）
重量：0.95kg
續航力：約7小時
端子：USB 3.0×2，micro HDMI，3.5mm耳機/米高風
售價：$10,980起
http://www.samsung.com/hk

評價：無論在外形或效能上都最接近MacBook，Geekbench的得分最高，並有更合理的擴充性能。但價錢比MacBook貴，屏幕又不及Retina來得順眼自然，鍵盤的空白、後退及輸入三鍵過小，容易出錯，可謂有利有弊。

實測結果

Gadget Guy之選

Steve：「我會選ATIV Book 9，因為較輕，方便出外工作。鍵盤按鍵過小不算太大問題，反而覺得回彈力不足，用來打稿手感一般。」

鬼輝：「Zenbook UX305比較抵用，價錢與MacBook Air相若卻有1080P高清屏幕，鋁合金機身完成度亦不俗。只是打開後有種台灣『娘』味，頂得順先好買。」
