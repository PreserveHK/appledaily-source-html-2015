EDC媾女開場白曝光
網民瘋狂造句抽水

大家如果有留意陳冠希（Edison）嘅IG，就會發現佢昨日個IG被一句「U seem like fun.」洗版，到底發生乜事呢？

是咁的。昨日希哥被爆喺IG同一個溫哥華靚女「Li」私信，仲一嚟就同個女仔講：「U seem like fun」一向媾女無數嘅希哥呢一句開場白，就被該女生公開咗，大批網民即發揮小宇宙，將「U seem like fun」瘋狂造句，創意無限﹗當然最基本係佢哋先喺希哥IG留言，寸爆佢又媾女：「U seem like fun」。

而除咗被「U seem like fun」洗版之外，網民仲發揮咗佢哋嘅創意，將呢句嘢改做其他字眼，包括：「U seem tree gun鍾樹根」、「U seem like TVB fun」、「易D生根」、「U seem like 梁美Fun」、「預先拉根」等等….
