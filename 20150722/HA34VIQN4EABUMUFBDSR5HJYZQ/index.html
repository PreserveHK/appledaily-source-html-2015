假匯豐電郵騙信用卡資料

【本報訊】昨日有市民接獲附有假匯豐銀行（HSBC）網站連結的欺詐電郵，騙徒藉詞提供免費信用卡驗證服務，藉機盜取信用卡密碼及身份證號碼等資料。事主因無法登入網頁避過受騙，匯豐證實事件並提醒客戶小心。

網域名不含「HSBC」

發現假匯豐網站的事主Anna，是匯豐e-banking客戶，昨晨11時，她收到「HSBC Internet Security Registration」主題電郵，內有英文及簡體字，稱為防範網絡非法交易提供免費信用卡驗證服務，建議登入超連結http：//www.hsbc.com.hk/1/2/home/service-secure輸入資料。
電郵上方有HSBC紅字及銀行標誌，Anna信以為真登入，但不成功，再發現電郵地址information438428@support.hk沒HSBC的網域名，於是致電匯豐查詢，終揭發是假電郵。本報其後登入可疑電郵的超連結，去到HSBC Internet Security Registration頁面，輸入信用卡號碼後進入信用卡認證頁面，需輸入卡主身份證和密碼。
匯豐銀行昨晚證實這是偽冒匯豐假網站，提醒市民勿登入網站內的超連結，金融管理局稱正關注事件。
電腦保安專家賴灼東指銀行不會透過電郵要求客戶提供個人資料，或於網上核實戶口，市民勿經電郵內超連結或可疑視窗登入網上戶口。
