【新車落地】Mazda 3 2.0皮櫳寬敞　入門扮頂級

年輕車主想買新車但預算不多，選擇還是有很多，日本車性價比好，餘下的就只有品牌和車款是否是你杯茶。Mazda的車款一向主打年輕活力，而且售價相宜，是上車的可考慮之例。他們的Mazda 3 2.0 Core Edition版本，其實是較早之前頂級版R-Grade的簡約版。稍稍將售價調低，但當然將一些豪華設備，例如Bose喇叭、Bi-Xenon頭燈、LED日行同尾燈和電摺鏡等等抽起。儘管作出這樣的改動，可是內櫳其實並非一般入門級。因為它依舊能給車主不俗的豪華感，座椅採用皮革物料，而並非絲絨。另外皮革軚盤更縫上紅線，熱血又活力。在我看來，已經是充滿誠意了。至於內櫳空間亦見寬敞，後排座位的腿部和頭頂有預留足夠空間給乘客，加上有天窗，更能成功增加空間感。

車身線條依然保持着Mazda「KODO-魂動」的設計概念，車身塗上紅色加上圓滑流線的車型，即沒有很驚為天人的造型和菱角，都能感受到這台車的動感。可惜，軚軨由18吋變為16吋，略為有點失望，車主可能要自己換軨了。
然而，驚喜位在於動力心臟，並沒有因為是入門版要轉引擎，依然是2,000c.c.直四SKYACTIV-G，可輸出165匹馬力，當到4,000轉時其210Nm的峯值扭力可發揮，低扭雖然不強，可以直路加速的表現確實帶爽勁。不過起步需要配合+/-撥片才能有較佳的效果。雖然是個少許的缺點，可是我覺得依然滿有駕駛樂趣的。

引擎︰1,998c.c. 直四SKYACTIV-G
馬力︰165ps / 6,000rpm
峯值扭力︰210Nm / 4,000rpm
售價︰$215,500

記者點評︰
讚︰入門版一點都不入門，豪華和活力感依然健在。
彈︰軨變為16吋有點失望，整體觀感遜色不少。
