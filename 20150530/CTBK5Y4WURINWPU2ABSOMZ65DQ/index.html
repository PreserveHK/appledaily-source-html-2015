【科技籽】Google I/O：搶先睇Android M六大功能

一年一度的Google I/O 2015又再舉行，Google同樣有不少新搞作，當中新系統Android M成為焦點，有不少創新設計，整理後有以下六大功能。

1.自訂Apps權限
如果你抱怨Android系統安全程度不足，新加入的自訂Apps權限應好滿足，用家可以自訂給予apps的權限，從而限制apps獲取用家資料。

2.Apps內都用到Chrome
當使用其他apps，點選網頁連結時，用家可以直接在apps內使用Chrome瀏覽器，甚至連內存的密碼都可直接使用。

3.Android Pay
聽個名就知是以Android手機作認證的付款系統，有機會取代信用卡嗎？

4.指紋辨識
系統直接內置指紋辨識功能，透過指紋辨識，可行解鎖、Android Pay付款及Play Store購物等功能。

5.更長電力
系統會跟據你的睡眠習慣，在用家休息時，自動減少資料同步的次數，從而達到省電的目的。

6.支援USB Type C
USB Type C已是大勢所趨，除了可以快速充電外，更可以變身尿袋，幫其他機種充電。
