【寵物食堂 】雞肉乳酪曲奇　色香味俱全

現今很多主人都喜歡為愛寵下廚，任職餐飲服務的郭恩彤 （Jackie），更喜歡為愛犬阿Bu炮製特色小食。即使平日工作忙碌，Jackie仍願意花時間上網找健康食譜，「我的狗狗比較饞嘴，而且腸胃很敏感，所以選用食材要很小心。」Jackie今天便會教大家整雞肉乳酪曲奇和乳酪士多啤梨蛋糕。兩款小食的材料中，除了乳酪有助腸道健康外，Jackie也選用了比低筋麵粉和普通麵粉高纖的黑糯米粉。為讓愛寵食得健康，Jackie更會花心思裝飾小食，她特意在美國網站訂購寵物專用的乳酪粉（Yogurt Royal Icing for Pets），混合自製的水果粉，為阿Bu整色香味俱全的雞肉乳酪曲奇。

雞肉乳酪曲奇材料：

1. 雞肉 50g
2. 雞蛋 1隻
3. 黑糯米粉 150g
4. 純乳酪 20g
5. 寵物乳酪粉 適量
6. 水果粉 適量

雞肉乳酪曲奇做法：

1. 首先打蛋，再加入純乳酪攪勻，然後分兩次加入黑糯米粉攪勻
2. 加入雞肉攪勻。Jackie建議先去除雞肉所有脂肪，比較健康
3. 所有食材攪勻後便可搓麵糰，搓至不黏手便可
4. 將麵糰壓平至1cm厚，便可印模，以180度焗20分鐘
5. 用熱水開溶水果粉。水果粉可以自己風乾水果後磨粉，或者買無糖無色素的水果粉代替
6. 加入適量寵物乳酪粉，溝成糖霜
7. 焗好曲奇後便可用糖霜做裝飾，最後待曲奇上的糖霜風乾便完成

乳酪士多啤梨蛋糕材料：
1. 鷹嘴豆粉 30g
2. 角豆粉 10g
3. 雞肉 20g
4. 純乳酪 20g
5. 雞蛋 兩隻
6.士多啤梨 兩粒

乳酪士多啤梨蛋糕做法：

1. 首先打蛋，再混合鷹嘴豆粉和角豆粉攪勻
2. 攪勻後加入純乳酪及切碎了的士多啤梨
3. 倒入蛋糕杯後，以170度焗20分鐘後
4. 於蛋糕面放上雞肉絲和士多啤梨碎便完成
