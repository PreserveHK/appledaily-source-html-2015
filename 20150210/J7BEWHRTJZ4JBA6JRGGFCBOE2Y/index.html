2K曲屏G Flex 2　更弧更纖更屈機

自從曲面屏幕面世，坊間不少人都認為出電子書才是王道，原因簡單，屈得又可以捲起，方便攜帶！不過，去年Samsung、LG分別推出Round和G Flex兩部曲芒手機後，大家又會諗，曲屏技術，只不過是純賣相、欠缺實際功能的小玩意。無論大家怎樣看，LG一於少理，再推出第二代G Flex 2，屏幕更弧、機身更纖，同時搭載時下最強2K畫質，水貨都要高開$6,580，玩新技術，代價不菲！
記者：陳國輝
攝影：譚建章

究竟曲屏賣相能否引起一群手機餓狼的換機意慾？外形暫且不提，皆因重點或來自高昂的售價！就以二○一三年公佈的第一代G Flex為例，當時售價就超過六千港元，同時，韓國另一巨頭Samsung的Galaxy Round，水貨更一度賣到七千港元以上，比128GB的iPhone 6更進取；至於另一顧慮或許是放售的價位，有先達sales說，LG旗下尤其是高階機，無論定價和收機價往往極速插水，像第一代G Flex現在的行貨價錢約$3,800，水貨更悲，兩千元有找。既然玩得貴價LG，就要有心理準備。

修身手感更佳

由N年前的Nokia「蕉仔」手機已可確定，彎曲的機身設計的確有一份魅力，而LG G Flex由第一代過渡到第二代，也許你認為彎得唔夠勁，亦未能突出曲屏的實際用途，但由舒適度先行的話，那就切合現實情況了。G Flex 2緊貼iPhone 6 Plus的5.5吋芒及7.1mm厚度，然而重量卻輕盈接近一成，僅152g。此外，G Flex 2也比上一代修身，長闊度稍作調整，再配合更彎曲的弧度線條，拎起手、傾電話感覺更貼面，放入褲袋亦舒適多了，由於外殼具備自我修復的塗層，大家也不用擔心會出現刮花問題。

新介面　配最強處理器

若對手機功能有點要求，應該對數據相當敏感，這方面，G Flex 2肯定滿足到大家的願望，皆因它是第一部配置最強處理器Snapdragon 810的八核機種！此技術在網上早就流傳，但正式拎上手試玩，感覺又如何？記者嘗試不斷打開功能視窗，若是一般手機，不久就會出現遲緩反應，然而新處理器除了不易發生delay、彈app甚至死機問題之外，更重要是實際操作方面，可以更爽快和得心應手；唯一擔心是沒有3GB RAM，運行一段日子，難免有點影響。

雷射對焦　快拍無起渣

鬥像素，似乎沒多大意義，反而在影相功能上增添特殊玩法，令操作、畫質表現更好，那不是更合心意嗎？從數字上，G Flex 2只配備1,300萬像素鏡頭，相比某些已達2,000萬像素的日系機，並不算突出，但此機卻內置一個叫Laser Auto Focus的對焦玩法，簡單講，即利用雷射光線探測物件與鏡頭之間的距離，從而捕捉影像，好處是能夠在低光環境下，拍出更靚照片。記者晚上九時到某商場戶外拍攝試相，值得留意是，對焦時並沒因環境昏暗而發生時清時模糊的問題，速度爽快；而照片質素更可形容為滑溜兼沒有起渣，光位分佈好平均，絕對抵讚。

記者長測　愛曲屏畫面

記者肥通和Steve首度拎起G Flex 2實機，為大家報道第一身試用感受！愛玩土炮的肥通，撞正外形先行的Steve，性格南轅北轍，究竟大家拎起部G Flex 2時，有何不同體驗？

肥通：咪又係大芒機，而且闊度得5.5吋，有乜咁潮先？

Steve：曲面屏幕design，傾電話就夠晒貼面喇，你睇！

肥通：又係曲屏！咁同上一代又有乜分別？

Steve：畫質已經好唔同啦！上代只有720P，令次已經採用2K，畫面sharp到不得了。

Steve：講到尾，我仲好鍾意佢有自我修復功能！

肥通：既然係咁，不如等我坐吓先，整花都唔使驚啦！

評分

肥通：★★★☆☆
Steve：★★★★☆

有片示範　刮極唔花

外國流行將新手機作極限測試，玩到你殘，而對於宣稱有自我修復功能的G Flex 2，當然無法逃離魔掌！有老外拍短片把手機和鎖匙放於同一褲袋，然後搖搖搖、再跳跳跳，如此伎倆，當然難不倒G Flex 2；老外再用鎖匙在機身上不停大力刮，結果，外殼真的出現些少花痕，但只要用手指捽一捽，又擦掉了。
