普立茲女作家手稿　埋藏半世紀後重現
美名著《怪屋疑雲》出續集

美國著名女作家哈珀．李（Harper Lee）一九六○年的成名作品《怪屋疑雲》（To Kill a Mockingbird）後就一直沒新作，出版商Harper前天突然宣佈，將於七月十四日為她出版續集，而新書是她逾半世紀前的舊作，只是手稿最近才被發現。

八十八歲的哈珀．李憑《怪屋疑雲》獲得普立茲獎（Pulitzer Prize），該書透過小女孩角度看一宗黑人蒙冤的案件，反映三十年代美國種族歧視問題，是美國學生指定讀物，一九六二年更被拍成電影。該書迄今賣出四千萬本，連英國球星碧咸亦愛讀，女兒Harper Seven正以作者的名字命名。

印二百萬本　幾小時內爆定單

新書《Go Set a Watchman》，講述《怪屋疑雲》女主角斯考特二十年後長大成人，回鄉探望當年為黑人挺身抗辯的律師父親，印證兩代價值觀的差異和轉變。她是在一九五○年代先創作了《Go Set a Watchman》，編輯看後未付印，覺得女主角的童年情節更吸引，要求她先寫「前傳」，遂以阿拉巴馬州鄉間見聞寫成《怪屋疑雲》。她一直以為《Go Set a Watchman》原稿已失，直至去年秋天才被發現。首批新書預計印二百萬本，消息一出，幾小時內已爆定單。
美聯社
