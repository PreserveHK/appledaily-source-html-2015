監控新招深圳住戶被貼QR Code

監控新招
深圳住戶被貼QR Code

內地維穩又有新招？深圳有民眾近日在住所門口，發現一張由「深圳市社區網格管理辦公室」的標籤，上面詳列房屋地址、房屋編碼，並且有一個QR Code（圖），掃瞄後更會顯示電話號碼。這名民眾稱「感覺像科幻版的納粹初期，給每家猶太人門上做個記號」。然而，這個標籤是配合當局的網格化模式，方便各部門收集訊息、排查問題，作嚴密的監控緊盯。
有網民在網上張貼照片後，其他網民稱包括北京、廣州，甚至甘肅蘭州都見過類似的標籤，馬上感覺身邊的圍牆、門禁系統、身份證、手機等等都可能成為當局的監控工具，「你無處可逃」。
內地法制網曾報道北京東城區的網格化社會管理模式，這系統基於「人、地、物、事、組織」，將全區的205個社區分為589個社會管理網格，網格內有管理員、助理員、督導員、警員、黨支部書記、司法力量等六類人員，其中助理員來自整合後的各部門協管員，負責訊息收集和問題排查等。
分析認為這樣的密集監控組織，是與當局收集到的大數據相輔相成，標籤不但方便職員每日製作訊息報告，甚至進行現場軟禁式控制，侵犯人權的監控盯防無處不在。
