【科技籽】Samsung Note 5曝光　不可插卡換電

Samsung即將於下星期在美國紐約發佈全新的Galaxy Note 5及S6 Edge，早前已不斷流出兩部機的規格甚至機身的正面照片。最近更有外國記者率先取得Galaxy Note 5，根據外殼所知，Note 5將不能插卡及換電。

Galaxy Note 5與上季推出的S6外形上差不多，不過沒有曲屏設計，機身一體成形。USB插位並不是傳聞中的USB-Type C；而S-pen可以完全插入筆槽，相信是可一按彈出。其他改變包括，懸浮指令亦換上新版本，而且出現了instagram的圖示，相信可以讓用家改變設定。

Note 5的規格與早前流出的相約，4GB RAM、內置容量有32、64及128GB選擇，而電池容量則有3000mAh，不過實際規格還有待Samsung在下星期舉行的發佈會公佈。

記者：容逸富
