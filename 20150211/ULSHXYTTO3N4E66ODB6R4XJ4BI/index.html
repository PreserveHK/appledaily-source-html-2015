2018冬奧前　雪地上的夢想之旅

古語有云：「做人冇夢想，同條鹹魚有乜分別？」不單止做人，治國亦然。因為一個冬奧夢，二○○三年，韓國平昌市成為「2010冬季奧運會」最後三個候選城市之一。在第一輪投票中，獲得最高票數進入最後一輪投票，與溫哥華競逐主辦權。結果，輸了。峯迴路轉的結局最叫人氣餒，但韓國人選擇信守一個幾乎沒有人在乎的承諾，在準備舉辦冬奧的場地上，把當年寫進計劃書上的一個項目進行到底，深信有日會如願。結果，在追夢路上，同時燃點了好多人的夢。
記者：曾冠輝
攝影：梁志永

「當年的諾言，我們做到了。」在到埗首爾的第一個晚上，與平昌冬奧組委會的人員晚飯，席間傳訊部總幹事申武澈先生如是說。
二○○四年，平昌冬奧組委會舉辦了第一屆Dream Program，邀請世界各地（主要是沒有雪的國家）青少年到平昌體驗冬季運動，其間還安排教練教授冬季運動技巧，項目包括高山滑雪（Alpine Ski）、滑雪板（Snowboard）、花式滑冰（Figure skating）、速度滑冰（Speed skating）、短途速滑（Short track speed skating）、冰上曲棍球（Ice Hockey）及冰壺（Curling），推廣冬季運動之餘亦培育更多冬季運動員，讓雪花開遍地。十一個年頭過去，經已有超過千五人參與，分別來自六十八個國家，遍佈七大洲，其中有四十九人後來更成為了自己國家的運動員出戰冬奧。
Dream Program 2015剛於過去的周末圓滿結束，來自四十二個國家的一百七十二位青少年完成了十三天的冰雪體驗。香港首次有代表參與，他是一位傷健運動員。

從A字腳失敗　到勇奪銅牌

有冬季殘奧，Dream Program亦有傷健組。十五歲的鮑錐耀是香港其中一名傷健運動員，經常參與本地公開三項鐵人賽事，因而得香港殘奧會垂青，獲選為香港代表參與Dream Program 2015。鮑錐耀因大腦麻痺導致下肢痙攣，肌肉比常人繃緊，四肢控制有一定障礙，「第一堂滑雪，我連A字都做唔到。」A字腳是滑雪第一堂必學的基本技巧，用以減速及煞停，對常人來說無難度，卻是他的第一個難題。「以前跟家人旅行時都有見過雪，滑過雪地水泡，滑雪就第一次。」由於行動不便，鮑錐耀需要利用特殊的輔助工具才能滑雪，一般滑雪場鮮有提供，教練亦未必敢教。隨行還有物理治療師歐陽佩恩，「他們需要靠一雙outriggers（似鋤頭又似柺杖的工具），方便滑雪和在雪地上行走。而且大會提供了一對一教練教他們滑雪，我主要在滑雪以外照顧他的需要，可以放心讓他去上課，他上課時我亦可以學滑雪。」
在差不多兩星期的行程中，除了當地觀光及一些文化體驗活動，其餘大部份時間都在上課。Sangtae是鮑錐耀的教練，「起初其實幾困難，他做不到A字，上課時又經常自己走開，言語又不通，是個幾大的考驗。」從A字都做不到，到訪問當天已上中級雪道，「但他記性好，總是能夠記住動作，到中段很快就掌握到轉彎的技巧。」以為他遜人一籌，其實也有長處。在訓練的最後一天，大會舉辦的一場小型奧運會，我們看了一場健全組短途速滑的賽事，好些來自南美的學員滑得慢，但樂在其中，落後足足一圈依然面帶笑容堅持完成賽事；也有些全身專業裝備，明顯在本國也有接受訓練的卡塔爾學員，爭先一刻失足跌倒，隨即怒擲頭盔拂袖離場，手腳健全又如何，名次以外早見勝負。當日沒有敗在A字上，鮑錐耀最後奪得一面銅牌。

率先滑 2018平昌冬奧賽場

Dream Program明年依然繼續，直到冬奧在平昌舉行為止。二○○三年初敗之後，平昌奧組委沒有放棄申辦冬奧，屢敗屢戰，連續三次申辦冬奧，終於成為「2018冬奧」主辦城市。作為Dream Program的住宿地點及雪上項目場地的Alpensia度假村，亦已被選為跳台滑雪、越野滑雪、長橇比賽等雪上項目的比賽場地。相較於旅行團最鍾意去的龍平滑雪場，Alpensia明顯人比較少。滑雪場位於江原道平昌郡有「韓國的阿爾卑斯」之稱的大關山嶺，深藏群山之中，風景極佳，而且由於人少，頓覺滑雪道寬敞得多，滑雪初哥亦不怕與人炒成一碟。
一場來到，當然唔放過滑雪的機會。滑雪場共六條雪道，分初、中、高級，由初級開始，雪道斜度平均，最闊處超過廿米，想撞到人都難；中級較高，斜度大，多彎，都算過癮；高級雪道路直，斜度甚大，幾乎從最高處直滑到底。吊車站的工作人員相當親切，每次上落總是揮手說「阿new黑些唷」，滑了十轉，他就說了十遍。問駐場教練：「雪場的雪孰真孰假？」教練想了想：「八成假，兩成真吧。」假作真時真亦假，適逢今年暖冬，韓國降雪的日子其實不多，於是有人擔心2018平昌冬奧會否如去年索契一般春意綿綿，「科技幫到手！」冬奧組委會人員直言，場地早已有完善製雪系統。到訪時剛好遇上漫天飄雪，立即親身試滑，雪質算不錯，雖未如粉雪幼滑，卻仍然乾爽，腳踏snowboard，速度高，易掌控，跌倒又不會太痛，新人老手皆宜。滑雪場亦有教練可即場授課，部份教練能操英語，溝通無難度。
