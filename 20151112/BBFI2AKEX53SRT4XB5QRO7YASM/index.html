JJ樊少皇排期結婚
王菀之11.28嫁Eric So

昨日11.11是脫離盛女的好日子，36歲的王菀之（Ivana）和34歲的賈曉晨（JJ）於光棍節齊爆婚訊，JJ於明年初下嫁43歲樊少皇，娶得美人歸的他在網上晒婚照，JJ宣示主權留言：「蛋皇是我的人了！」Ivana跟Figure王蘇卓航（Eric）於本月28日結婚，婚禮從簡不擺酒不影婚照，Eric於今年金像獎舉行前夕求婚，率先獲男友頒最佳女主人獎，Ivana昨憑歌晒愛：「遇上你，太感恩，不遲亦不早。」

【蘇王聯婚】現年36歲的王菀之和Figure王蘇卓航拍拖兩年，昨日Ivana於光棍節在微博宣告脫離單身，藉自己歌曲《好時辰》的歌詞訴心聲：「『遇上你，太感恩，不遲亦不早』感謝天父，感謝父母，感謝所有好兄弟姊妹，感謝大家，也祝福天下有情人終成眷屬，白頭偕老，攞個彩先——Eric So, you're SO LUCKY。」

車上收戒指

兩人於上月22日到婚姻登記處入紙結婚，據知，Ivana趕做11月新娘，婚期訂於本月28日，惟因為男友藝術家脾氣為人低調，所以這天一對新人只進行簽紙儀式，不擺酒不影婚照，只會跟雙方家人食飯分享喜悅，溫馨地慶祝。
昨日Ivana甜蜜回應婚訊，她說：「4月19日係金像獎頒獎禮，但4月18日我同Eric喺韓國首爾忙緊佢嘅展覽，所以18號我一定要返香港。就喺去機場嗰程車，佢攞咗隻戒指同我求婚，當時心情好激動。金獎像有冇獎都未知，但就好想率先頒一個最佳女主人獎畀我。」Ivana表示兩人認識時是單身，因同樣愛藝術愛旅遊，大家價值觀相若，所以很適合對方。Eric則在社交網貼一對Figure公仔，多謝大家祝福。
Ivana和Eric的情路波折，有女星殺手封號的Eric，曾跟盧巧音、蒙嘉慧和杜如風拍拖，有傳當年杜如風和Eric情海翻波，Ivana攝位主動出擊冧掂Eric，Ivana對此曾呼冤否認。

杜如風高興

昨兩人公告婚訊，杜如風送祝福：「恭喜晒，有情人終成眷屬，白頭到老，執子之手與子偕老。」曾跟Ivana傳假緋聞的張敬軒說：「好開心，搞到我都想立刻宣佈婚訊，昏迷個昏，好高興一直睇住Ivana情海翻波到搵到啱嘅人，Eric係冇得頂嘅人，好人又體貼，Ivana會好幸福。」楊千嬅亦送恭喜：「戥佢好開心，祝佢同Eric白頭到老，早生貴子。」容祖兒也說：「戥佢哋開心，希望佢哋生BB，恨見到佢哋係矇豬眼BB。」 曾合作《金雞SSS》的吳君如則說：「好戥Ivana開心，搵到另一半咁愛佢愛佢，怕你攞唔到獎，用下半生𠱁你開心。」
