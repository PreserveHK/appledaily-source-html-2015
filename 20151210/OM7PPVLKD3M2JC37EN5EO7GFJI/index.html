【寵物食堂】愛寵有口福　DIY狗狗pizza燒賣

貓貓狗狗天生為食，就係人都知，但怎樣令愛寵有口福之餘又食得健康，你知道嗎？其實除了乾糧和罐罐，人類食物愛寵一樣有份，只要不含調味料及人工色素，細心挑選對寵物無害的食材，就可以為牠們炮製愛心美食。寵物籽找了幾間自家製小食店，一連四星期教大家整各款寵物小食，主人有空不妨一試，讓愛寵成為你的食評家。這次我們就請來李彥其（Yanki）示範整狗狗pizza和雞肉燒賣，無論材料和造法也相當簡單。Yanki指選用食材也很靈活，主人可視乎愛寵口味而選用不同材料。

狗狗pizza材料：

1. 比得包一個
2. 吞拿魚一罐
3. 三色椒，各取約1/8份量
4. 無鹽芝士20g
5. 日本南瓜半個
6. 切粒芝士 適量

狗狗pizza做法：

1. 首先將南瓜整成南瓜蓉，再將南瓜蓉平均塗在比得包上
2. 切去三色椒頭尾部份及去芯，取1/8切粒
3. 倒出吞拿魚罐頭的水份，把吞拿魚平均鋪在比得包上，再鋪上切粒的三色椒
4. 鋪上切粒芝士
5. 把焗爐預熱至170度，放入pizza焗10分鐘便完成

雞肉燒賣材料：

1. 上海雲吞皮 三塊
2. 雞柳兩條 50克
3. 青豆三粒
4. 小量水用作黏實雲吞皮

雞肉燒賣做法：

1. 把雞柳切碎，切得越碎越好，方便狗狗進食
2. 用手沾水，弄濕雲吞皮的底和面，然後把雞肉包在雲吞皮內，包雲吞皮時可沾水黏實對摺的位置，讓雲吞皮不會散開
3. 把青豆放在雞肉上
4. 隔水蒸10分鐘便完成
