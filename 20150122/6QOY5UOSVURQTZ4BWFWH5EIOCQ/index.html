渣馬前夕　運動配件戴住測

渣打馬拉松將於星期日舉行，相信各參賽者都已進入最後備戰。有細心留意的話，近年不少跑手都有使用電子配件裝備，如運動手錶、耳機、手環等都頗為常見，今次我們找來RunOurCityCrew的街跑成員實測這三大運動配件，跟大家來個賽前大檢閱。

記者：臧諾

攝影：劉永發

運動手錶
Polar V800

屬品牌的旗艦級產品，可連接電腦及手機上傳運動表現及路線，並製成詳細圖表。最特別是有Smart Coaching功能，能告知訓練負荷及效益，協助長跑訓練。

Adam點評：全面貼心

-
有齊圈速比較、上落坡度、身體負荷等，夠全面，還會建議休息時間，頗貼心。
-
配有金屬錶身，三隻之中最墜手。
-
只有英文顯示不太方便。

GOLife RUN 120+

台灣土炮，可偵測運動路程、跑速、步數等資訊，同樣可連接電腦上傳數據，勝在性價比高。

Adam點評：訓練拍檔

-
最特別是能自設訓練內容，如均速跑或間歇跑的訓練。
-
具備中文顯示夠方便。
-
訓練數據不夠Polar詳細，但對長跑訓練來說尚算足夠。

SONY Smartwatch 3

屬智能手錶，可對應Runtastic運動軟件，記錄步速、距離、卡路里消耗等基本數據。此外，亦有一般智能手錶的功能，能用來回覆WhatsApp及電郵。

Adam點評：功能取勝

-
外形較時尚，可作日常使用。
-
能下載app、傳送訊息，功能比另外兩款手錶多。
-
不過只能記錄最基本的運動資訊表現，不夠專業。

運動耳機

Monster × adidas Originals

大牌子的聯乘產品少不免玩設計，機身配有標誌性的三葉圖案，並以湖水藍作主調着實搶眼，還採用了Monster專利SportClip技術，聲稱可避免耳機在運動時滑落。

Wing點評：穩固聲靚

-
勾形軟膠設計令耳機非常穩固。
-
音質最靚，隔音不俗，因此不宜戴着街跑，以免發生危險。
-
感覺較重，跑步時會覺得有點累贅。

SMS Audio BioSport

SMS Audio與Intel的合作成果，具備防汗功能。內置生物識別光學感應器，可透過跑手的耳朵收集心跳率等數據，在RunKeeper app中展現出來。

Wing點評：街跑安全

-
內置測心跳率功能是最大賣點，可惜只對應指定手機，如iPhone及部份Android手機，有點局限。
-
隔音水平適中，較適合街跑用。

yurbuds Focus 300

此耳機擺明車馬主攻運動市場，特意加入U形耳勾設計及線控功能切合運動所需。品牌亦減弱了隔音效果，以便跑手留意路面情況。

Wing點評：輕身有線控

-
較為輕身，不覺有負擔。
-
三者相比之下，音色不算最好，但低音頗出色。
-
有線控功能，但按鈕太過細小，跑步時較難操控。

運動手環

Misfit Shine

體積與十元硬幣相若的運動追蹤裝置，可配上膠製錶帶隨身攜帶。只要輕拍裝置三下，就會開始記錄簡單的運動資訊，如步數、卡路里消耗等，晚上能自動偵測睡眠質素。

Roni點評：舒適美觀

-
外形較美觀、簡約，容易配襯。
-
尚算舒適，戴着睡覺應該沒有問題。
-
不夠靈敏，有時感應不到指示。

小米手環

裝置沒有多餘的設計，只有三個指示燈顯示用家每天的運動進度。可連續使用近一個月，它就會悄悄記錄你的運動及睡眠情況。

Roni點評：抵買之選

-
除了跑步，還可以記錄跳繩、仰卧起坐的運動量。
-
價錢非常吸引。

Jawbone UP MOVE

功能跟其他手環差別不大，都是用作長期記錄基本健康數據。UP MOVE的特點是可以扣在衣衫上，加上其續航力長達半年，頗耐用。

Roni點評：記錄清晰

-
可夾在衣衫，感覺輕便。
-
app用具體圖表顯示健康紀錄，比另外兩者清晰。
