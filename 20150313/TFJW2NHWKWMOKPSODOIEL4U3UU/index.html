親友拜祭　同事致哀

【本報訊】「Life is so fragile...My heart is so torn apart」（生命是如此脆弱...我是如此傷心） 兩名航空界新星殞落，一眾好朋友在facebook留言悼念，悲嘆人生無常；更有朋友上載與死者黃奕的合照致哀。
昨晚六時半，在警方安排下，兩名死者的近百親友，包括死者譚兆璋的父母，返回車禍現場獻花點蠟燭致祭。各人在跑車失事衝上天橋底的草坪上圍成一圈，中央放滿帶來的鮮花，低頭默哀禱告，其間有人傷心落淚。眾人逗留約一小時後黯然離去，譚的父母難掩喪子之痛，要其他親友在旁攙扶和安慰。

近百親友致祭

在兩人經常出入的香港機場，剛在美國返港的國泰外籍機師Tom表示，雖然在香港居住逾七年，但並不認識兩名意外身亡的華籍機師，對兩人如此年輕便去世感到驚訝和惋惜，他相信國泰機師工會內部稍後會在港舉行悼念活動，讓同事緬懷兩人的工作及生活點滴。
昨返回國泰城接受訓練的蔡小姐稱，同事從新聞報道中獲悉事件後，公司上下均感傷心。她表示雖然未曾與死者一同工作，但大家都是在本港航空界服務，故此對他們死訊感特別難過。
