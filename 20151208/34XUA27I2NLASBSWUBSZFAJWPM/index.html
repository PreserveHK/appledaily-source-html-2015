【古怪周邊】罐罐形喵CAM　掛頸睇主子之日常

所謂騙徒手法層出不窮！最近日本就有IT商店，以低解像Cam仔，設計出一款寵物攝像鏡頭。

設計上非常Low Tech，機身尺寸係超巨大的108 x 176 x 46mm，重量122g，比一般大家玩開的Action Cam大了近一個碼，不過騙徒非常聰明，機身做成主子們最喜愛的「罐罐」形狀，看相中受害喵星人，基本完全失去反抗能力！

據稱夾在頸圈之上，再插上microSD卡，就可以錄下主子生活之日常；畫質方面，動畫可達640 X 480，相片則可達1,280 x 1,024；不過最令人開心之處，是它內置有隔時拍攝功能，所以儘管只能拍下最長1.5小時的畫面，但其實裏面已包含了主子近3.7小時隱私生活。機價唔算貴，只需5,980日圓（約港幣$378）。

記者：朱崇德
