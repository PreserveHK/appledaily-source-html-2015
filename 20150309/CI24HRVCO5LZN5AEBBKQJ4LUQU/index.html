見手袋冒煙　即掟出車廂港鐵客　Note II爆炸

見手袋冒煙　即掟出車廂
港鐵客　Note II爆炸

【本報訊】行駛中的東鐵列車發生手機冒煙事故，中年女乘客昨乘列車往新界，途中發現手袋內三星Galaxy Note II手機冒出濃煙，即時按動車廂緊急掣求救，可是濃煙持續，她情急下趁車門仍然打開，將手機連同手袋擲出月台。事故中無人受傷，消防員發現手機電池發脹，機殼及機身殘留爆炸焚燒痕迹，懷疑手機電池過熱爆炸肇禍。
記者：張培生　梁瑞帆

涉事女乘客姓黃（四十一歲），她事後離開車廂向消防員遘述事發經過，猶有餘悸，但慶幸並無受傷。消防員到場調查肇事的三星Galaxy Note II手機後，不排除是手機電池過熱自焚致爆炸，連手袋亦被熏黑。
港鐵發言人指，整個事件歷時僅兩分鐘，事故中東鐵列車服務未有受阻，班次如常，女事主乘坐的列車其後亦繼續行駛載客。
事發於昨下午一時半，姓黃女事主搭東鐵落馬洲線列車前往新界，當時她把三星Galaxy Note II手機放進一個皮袋內，列車駛往火炭站途中，她發現手袋冒出濃煙，打開竟見手機正在冒煙，其間列車車門打開，多名乘客下車離去。黃女續發現手機冒出更多白煙，立刻按動車廂內的緊急掣向車長求助。可是手機持續冒出大煙，由於情勢危急，她趁列車仍未關門，立刻脫下手袋，將手袋連同冒煙的手機擲出月台。
其後有車站職員前來協助，他們在月台發現事主的手機及手袋，當時手機機身與外殼已經分開，手機電池更脹大幾倍，至於手機機身、機殼及手袋均殘留熏黑痕迹。港鐵職員找到女事主，立刻陪同她離開列車，在月台向她了解事發經過。

疑電池過熱自焚

港鐵同時報警求助，消防員接報趕至月台現場了解事件，初步懷疑手機的電池過熱致自焚，不排除曾經爆炸，事件無可疑。女事主事後取走手機殘骸離去。至於事發列車不需疏散乘客，事故後列車繼續行駛載客。
今次手機電池自焚爆炸事件，女事主及時將冒煙手機擲出月台避過一劫，城市大學電子工程學系副教授鄭利明指，若當時手機在港鐵車廂內爆炸，後果不堪設想，可能傷及更多乘客，不過幸好昨日是假日，火炭站亦人流不多，未有乘客遭擲中受傷。
他補充，現時手機電池多數由稀有金屬鋰（lithium）製造，當鋰電池着火焚燒，絕不宜用水淋澆撲救，因為鋰金屬遇水可能會爆炸，甚至釋出更多有毒化學氣體，危害更大。他續稱，其實手機被投擲着地一刻，鋰電池可能遭到猛烈撞擊，導致電池內的物質產生劇烈化學反應，若撞擊力猛烈足以釀成爆炸。
