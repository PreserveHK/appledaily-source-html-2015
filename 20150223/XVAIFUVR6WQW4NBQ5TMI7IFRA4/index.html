創業作業：女programmer - 宋漢生

前排，部Kindle無端端多了兩本我無買的書。頭一本取消到，尾一本，找不到簡單方法取消。重整案發經過，我估大概是，我個女攞我部Kindle玩，部機無設密碼，她見掣就㩒，㩒下㩒下就幫我買了兩本書。一個不識字的小朋友，亂㩒之下，都可以買到書，可能是做user interface的最高境界，如果Jeff Bezos知道，應該會笑得好開懷。
最絕的，是錯買的書，不是隨機，因為能夠在Kindle顯示出來㩒得錯的，都是Amazon系統估我最有可能有興趣的書，衰在又估得真係好準，尤其是我有這麼多往績在它手中。那不易退的，叫《The Innovators》，屬於我無打算買，但錯買了又會睇的一類。
本書作者，前作是Steve Jobs的傳記，之前的Einstein和Ben Franklin都好睇，這本《The Innovators》，同以前的作品最大分別是，名人傳記，側重個人的成就，這本書，講的是電腦發展史，橫跨百幾年，穿插幾十個人物，着重的是集體的功勞。

過往軟件工作屬低檔

當中的故事，在好多其他書出現過，這本書基本是炒埋一碟，其中比較有趣的，是在幾個故事之中，見到女性的影子。史上第一programmer好出名，是個女人，叫Ada Lovelace，老竇是大詩人拜倫，私生活出名那個，Ada阿媽為了令個女不走老竇的藝術路，送個女學數學，結果在電腦發明之前一百年，已經寫了史上第一個程式。
未有電動的computers之前，computers指的是人肉計算員，絕大部份是女人。後來二戰催生了第一部現代電腦，第一批programmers，清一色女人，理由好簡單，男人覺得男人應該搞硬件，軟件這類低檔工作，當然是交由女人負責。再過幾十年，有畫面的個人電腦誕生，發明的Xerox影印公司的高層，覺得鍵盤似打字機，而打字這類工作，當然是由女秘書做，不屑一顧，結果拱手把市場讓給蘋果和微軟。
自己團隊，programmers當中，有兩個女生，佔大概百分之十，今時今日，整體業界都是差不多比例，點解女生比例低，不會是因為歧視女生，人才本來就難得，仲要理是男是女就太蠢。業界比例低，是因為讀相關科目的女生比例本身就低。點解揀這科的比例低？再追問落去，越數越早，隨時由一出世父母送甚麼玩具，都可能有關。有位女性朋友，開了間教小朋友寫程式的學校，叫First Code Academy，我估，越多女孩接觸到編程式的樂趣，未來的startups，會越來越少悶死人的和尚寺團隊。
