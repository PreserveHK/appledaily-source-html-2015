【寵物學堂 】貓咪也扮靚　雙蝴蝶結飾物夠晒cute

本來毛茸茸眼大大的貓狗已經夠可愛，加上主人為牠們悉心打扮，cute上加cute是常識吧。寵物籽找來不同的手作人，教大家DIY寵物飾物，為愛寵打扮出獨有style，要型要可愛要高貴都有！三位少女章雅媛（Nadia）、譚詠雯（Wing）和蔡倩珩（Alison），不僅自己愛打扮，就連為寵物打扮都花盡心思，Wing更指見到愛貓戴着自己一針一線縫製的飾物，有滿足感之餘，愛寵被like爆不知幾開心。除了幫寵物扮靚，她們也注重寵物飾物的安全性，選用的頸繩配有安全扣，如寵物玩耍時頸繩受力拉扯，安全扣便會自動鬆開，以防寵物受傷。這次她們會教大家整三款蝴蝶結寵物飾物，主人落足眼力學嘢啦！

雙蝴蝶結頸繩

材料：
1. 粗絲帶、幼絲帶
2. 鈕扣
3. 安全頸繩
4. 線

做法：
1. 首先因應蝴蝶結大小度好所需絲帶的長度，剪出兩條粗絲帶及一條幼絲帶，Wing建議一個蝴蝶結長度約用25cm絲帶
2. 將剪裁好的絲帶繞圈，把頭尾部份用膠水黏實
3. 交叉叠起兩條粗絲帶，再將幼絲帶放在中間作點綴
4. 用線繞在中間，把三條絲帶綁在一起，固定位置，再用幼絲帶包裹線位，用膠水黏實
5. 用鉗去除鈕扣後凸出部份，使鈕扣背面更平滑，方便固定位置
6. 用熱熔膠槍把鈕扣黏在雙蝴蝶結中間
7. 用膠水把雙蝴蝶結黏在安全頸繩上便完成
如果想有更豐富的視角效果，可在雙蝴蝶結下加一層喱士，看上去既可愛又高貴。材料比雙蝴蝶結頸繩多一塊喱士布就可以。

雙蝴蝶結喱士頸繩

做法：
1. 剪出所需喱士的長度後，每2至3cm對摺，反覆對摺約5至6層
2. 用針線縫起喱士頂部
3. 用熱熔膠槍把喱士頂部黏在雙蝴蝶結中間
4. 喱士黏在雙蝴蝶結後，喱士左右兩邊是垂下的，把兩邊喱士拉起，黏在雙蝴蝶結底邊，便會造出傘形效果，更加美觀

雙蝴蝶結百寶袋

材料：
1. 粗絲帶、幼絲帶
2. 安全頸繩
3. 線
4. 喱士布
5. 鈴鐺
6. 紙樣
7. 格仔布

做法：
1. 雙蝴蝶結做法同上
2. 做百寶袋，首先要畫好所需大小的百寶袋紙樣，然後放在格仔布上，用粉筆畫出百寶袋形狀
3. 畫好後將格仔布對摺，對摺時要預留2cm空位在頂部，接着沿圖形線條剪出百寶袋，裁剪時切勿完全貼着粉筆線剪，在粉筆線外圍預留約2cm空位才剪
4. 用衣車沿粉筆線縫好，縫好後，在預留的空間將格仔布從內向外反，就會形成一個百寶袋
5. 度出要用的喱士，剪下並黏在百寶袋上，把頭尾多出的喱士收入百寶袋兩邊的空位
6. 將頸繩穿入百寶袋，用衣車將頸繩和百寶袋縫起，固定位置
7. 最後用熱熔膠槍將雙蝴蝶結黏在百寶袋上，就大功告成了
