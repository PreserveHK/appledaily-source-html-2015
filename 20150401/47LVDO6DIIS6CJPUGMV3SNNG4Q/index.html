【文化籽】輟學藝術家Simon Birch：閱歷比學歷重要

【文化籽：藝文沙龍】

香港學生輕生事件時有發生，上月巴塞爾藝術展期間剛與年少輟學的藝術家Simon Birch談起這問題，沒想到數天後又出現學生因功課壓力而自尋短見的新聞，令人感慨。Simon Birch居港十八年，從沒有學歷的外籍勞工到戰勝癌症的著名藝術家，他的人生比過山車還要高低起伏。多年來他從沒有為學歷低而苦惱自卑，不是因為經歷過生死關口才有所頓悟看輕名譽、也不是因為他成功了才輕視學歷，而是他一直相信，「你的人生閱歷才是真正的學歷，並非那一紙證書。」

「香港教育太過非黑即白，彷彿有好成績就是成功，不然就是失敗。但生命並非這樣，學歷或成績只意味着你通過某些測試，或許代表有較好的生活質素，但並不代表你有資格做好一個人。真正重要的，是那些不一樣的經歷，冒險、戀愛、幫助人，這些事情一紙證書沒有記載，卻令人變得有趣。香港社會告訴你的人生，並非是開心的生活。」
Simon坦言自己或許不夠資格談論教育問題，但在這城市生活多年，他覺得很多香港人對成功定義仍很狹窄，「這個社會定義人怎麼成功，怎麼穿衣服，如何挑選好車，這令人感到困惑。人生不是讀好書做好工作，年輕學生因學業壓力而自殺，作為大人和社會都應該反省。學術上的成功令人感到壓力，人的成長失去了個性。當談到教育時，需要再重新思考。

從$500賣到$100萬

他沒有學歷包袱，因為最精采的學歷就是他的人生經歷，而Simon也確實為他的經歷增添傳奇。他成長於倫敦郊區小鎮的貧窮家庭，自小便討厭讀書，經常走堂、考試成績不理想，最後更被趕出學校。如果僅以考試成績或學歷來衡量人生，那他或許是失敗的。他熱衷於飆車和DJ文化，反而對讀書不以為然，「在我成長的地方，年輕人並不十分嚮往讀大學。」二十歲那年他離開英國，輾轉到多個地方遊玩，最後在香港回歸那年落戶這座小島。
那時他身無分文，沒有學歷沒有專長，不遠萬里前來卻沒有任何打算，以今天標準來衡量早被定義為「廢青」。那可以做甚麼呢？勞工。「那時在青馬大橋做高空工作，下雨時不用返工，可以畫畫。」這對當時的他來說是種小確幸，皆因他自小就喜歡看《蝙蝠俠》、《超人》，一直想成為漫畫家，但打工的生活只允許他利用空餘時間畫畫。後來政府更改政策，Simon隨即失業，之後一段時間靠做DJ搵食，同時堅持着畫畫的熱情。後來DJ工作也因政府政策而停止，再度失業，想過回英國讀藝術學校，奈何又是學歷與資金的問題，「沒有學歷，我便舉辦展覽來建立自己的履歷。」懷着滿腔熱誠，他逐一拜訪畫廊想舉辦展覽。香港是個現實社會，當時沒有知名度的他自然屢吃閉門羹。
費盡九牛二虎之力，親力親為租場做展覽，展出一系列畫朋友的畫像，最後卻一幅也沒有賣出。這聽起來多少讓人覺得沮喪。幸好那時開始有人邀請他畫人像。二○○三年，在香港畫了六年畫後，終於賣出第一幅畫，售價是五百元。數目雖小，卻讓他看到未來道路怎麼走，「很驚訝有人買我的畫，這給我足夠的動力去做更多的作品。」之後雪球越滾越大，現在他的大幅作品的售價已逾百萬。
按邏輯，故事自然往好的方向發展，畢竟Simon總算挨到出頭天，可惜他是個喜歡冒險、喜歡嘗試的人。在畫畫的框架外，他也開始做策展，做一些非商業性的實驗項目，影像、雕塑、裝置等，從二○○六年的《Outside Context Problem》開始，他幾乎每年策展一個項目，雖然燒錢，卻為他帶來了很多合作的夥伴，他也這樣慢慢建立起非正式的履歷。唯一真正的學歷是二○○八年在香港藝術學院修畢藝術碩士課程，他坦言一方面想了解教育，另一方面是想在風格上走前一步。
這一年他被斷定有癌症，Simon以為自己很快會死，但這痛苦的經驗卻令他獲益匪淺，「當人接近死亡，所有事情都變得不重要，是否成功出名、擁有金錢名車與否都沒有意義，你很清楚地看到這個世界甚麼最重要。這段時間有很多朋友幫忙，讓我感受到愛。這種經驗令人清晰地看到人生，這是很有價值的。」之後他更積極地分享愛，為香港癌症基金會作慈善拍賣，去幫助其他人。
藝術來自生活，回憶起這一段段經歷，Simon說自己的藝術就是來自之前那些生活的林林總總，「青馬大橋的經歷讓我學懂合作，癌症的經歷令我面對一個新的世界，我的每段經歷，都成為我畫筆及展覽的靈感，生活的元素轉化為藝術。」兩年前他曾在中環Ben Brown畫廊舉辦過《Hooligan》，畫中人物秉承他以往作品的風格，動感而帶點暴烈，作為上次展覽的延續，這次作品依然充滿戲劇效果，畫刀下色塊和諧地重叠着，依然抽象，人物則更趨模糊，「展覽叫《Inevitable》，就像命運，意味即將發生的事情。」新作品給人的感覺是被切開了，紊亂而無迹可循，他說這是新的方向。

居港18年　「主流認為我是鬼佬」

居港十八年，Simon早是香港居民，但在很多港人眼中，他依然是異鄉人，「有些人會接受我是香港人，但主流社會仍認為我是鬼佬。我最近有個項目在紐約，那裏歡迎任何人，住在紐約就是New Yorker，香港不是這樣，社會裏有劃分。我一半人生在這裏度過，我不局限自己的身份，屬於哪個城市哪支國旗並不重要，我感興趣的是人，我緊張的是我是否一個好人。同樣，香港藝術家的身份並不重要，重要的是藝術家。」
在香港創作多年，他坦言要成為藝術家很難，「因為沒空間、沒金錢、沒支持，如果你是年輕藝術家，你只想畫畫，你日頭需要找份工作來支付租金，香港沒有便宜的租金，這意味你作為藝術家的時間很有限，這很難令你成為藝術家，也很難說服你的家人讓你追求藝術。」人們經常說香港是個文化沙漠，他認為這並不正確，「香港早有藝術的種子，問題是沒有人灌溉，香港某程度上沒有成功地支援、鼓勵他們，如果一座城市只是有商場、房地產和遊客，這有點無聊。」最近有許多藝術活動在香港發生，Simon對此則比較樂觀，「它給港人帶來的訊息是藝術文化或許是一件大事情，我們應該參與更多，長遠來說對香港是好事，我們會出現好的藝術家。」他舉例去年雨傘運動期間，整條街上的人都在做藝術，「這是很美的東西，人們很好地表達自己。藝術與正義相連，尤其是社會不安的時候，我相信藝術是其中一樣啟發人們的東西，允許人們表達自己。」

居港外地藝術家談香港

香港有不少外國藝術家居住，我們找來三位現居香港的藝術家，分別是來自英國的Norman De Brackinghe、荷蘭的Simone Boon及加拿大的Peter Yuill，談談他們眼中的香港。

問題：
1.為何想留在香港？喜愛甚麼東西嗎？
2.你是全職藝術家？之前做過甚麼工作？
3.你覺得你是香港藝術家嗎？
4.你的廣東話程度？

Norman De Brackinghe（1976年起居港）

1.
香港有很多不同國家的人，且文化間能互相影響；我喜歡香港食物，中華文化很吸引我。
2.
不算是，我曾經在出版公司工作，也負責設計和平面藝術相關工作。
3.
我會說自己是香港藝術家。我住了近四十年，這裏是我的家、我的家人在這裏，我做的很多東西都受這裏影響。
4.
我只會說非常簡單的，但聽起來非常糟糕，不過我有努力學習中國歷史。

Simone Boon（2004年起居港）

1.
香港東西文化混合跟我很像，感覺像在家一樣；我喜歡維港的天際線及西貢漁村的小街。
2.
我曾做過小學老師及荷蘭文老師；家中經濟主要由另一半負擔，令我可以做全職藝術家。
3.
我畢業於藝術學院，這令我成為本地藝術圈一員。由於不是這裏出生成長，相對土生土長藝術家，我又不完全是香港藝術家。
4.
我不會講廣東話，在香港你不需要懂。

Peter Yuill（2010年起居港）

1.
香港給人的感覺繽紛多采，安全、乾淨，交通很方便。我喜歡舊香港文化，唐樓、霓虹燈、小店、城市的雜亂，喜歡在小巷中穿梭。
2.
我的全職是做藝術，但不止一份工作。我做過很多工作，包括高爾夫球場、貨車工廠等。
3.
算是也不算是。我居住香港五年，覺得自己是這城市的一分子。但作為藝術家，你不只是地區性的，應該是全球性。
4.
我的廣東話其實不差，只是口音有點怪。
