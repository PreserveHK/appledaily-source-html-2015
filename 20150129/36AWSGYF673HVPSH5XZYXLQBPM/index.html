港式市集有態度

香港人的確喜歡跟風，像近來遍地開花的港式市集。何謂港式市集？我想起了天光墟。且看今天的城市，各區一式一樣，幾乎每條街都是大大小小的連鎖店。我訪問了一位在市集代客寫詩的年輕人和一位在天光墟擺賣的老檔主，他們不約而同地慨嘆「日做夜做？為乜？」一個在日出前擺賣，一個在日出後擺檔，二人素未謀面，關係卻是微妙錯落。年輕人不鼓勵消費，要說故事；天光墟的檔主無非都是吹水為樂以度日，建立一種人情味。然而，這群參與市集的年輕人，是一群醒覺者，不斷探索生活的態度，天光墟檔主的想法較簡單，但其實都在抗衡社會主流的價值。正如市集零零碎碎地散落在工廈的攝影室和天台，天光墟在不見光的天橋底，它們活像一隻爆牙，被逼出來，細小，奇形百狀，卻是絕處逢生。

記者：陳芷慧

兩代人的擺檔生活

這兩天「代客寫詩」檔主王道顯在牛頭角nomad nomad工廈天台市集擺檔，不消費冇所謂，只是代客寫詩，大前提是以故事來作等價交換。由中午至晚上七時，人來人往，其實有點像茶樓搭枱，有人坐下，有人離開。大家素未謀面，卻曾經圍在一起，又曾聽過對方心裏的故事，久不久又會有一兩句搭訕。有無數話語，把兩者櫳底沉靜已久的回憶翻出來。晚上他在床上整頓思緒：「這兩天是一種新刺激，像一口微辣的蘋果酒或鬧鐘，叫人找回生活中讓人感動的地方。」明天星期一又上班，他關燈，睡了。
#　　#　　#　　#
星期一的凌晨五時，深水埗通州街玉石市場外的天光墟，只有寥寥數檔。六十多歲的檔主「光頭佬」，昨晚七時至凌晨三時剛在郵輪「雙子星號」當帶位，下班後換上一套短衣運動裝，掩飾疲憊的身軀來到天橋底，將存放於露宿者「家門」外幾袋尼龍袋翻開來，一堆堆舊衫褲、煲蓋、一包雲耳等貨品鋪滿整個空地像紅地毯。一位中年男人拿起袋裝一百個小蠟燭問價，光頭佬粗聲粗氣地答道：「二十蚊一百個蠟燭都要講價？」日復日對着同一群格價的街坊，還有一些一晚來回數遍"window shopping"，又或蹲在這裏挑上幾小時。唉！為乜？他自問自答：「冇計！搵食艱難。」

上一代為生計，這一代為生活，想學人講態度，哪有時間？

日復日工作差不多，有時候，道顯會自問，勞動的本質是甚麼？他今年二十六歲，現在於藝術機構工作。每天有開工冇收工，三年間每天死磨爛磨，他每晚看得最多都是街燈照着深宵車輛飛過，偶爾憶起小時候走過大笪地的風光，一個個電燈泡搖來搖去，印象中有人歌舞有人笑。現在？似乎都加班慣了。很多美好的人和事只在記憶中浮浮沉沉。因此，他特別珍惜生活中讓人感動的地方。去年平安夜，隨意報名參加油麻地活化廳的地攤市集，學着天光墟婆仔「賣剩蔗」，帶了玻璃樽、同事儲下的酒塞、拍紙簿等貨品，期望自由定價，可惜無人問津。
#　　#　　#　　#
光頭佬想起廿年前從大陸來港的事，「算一算，今天剛好在深水埗擺檔廿載。」他說話時帶着一種看透世情的語調，「以前我在鴨寮街擺檔，一日賺千多元。現在？最多只賺百多元，靠這個檔口養活？真係仆街！」鄰檔的黃伯聽到也笑了。每日百多元，對他而言是花紅。一位街坊站在黃伯的貨品堆中氹氹轉，轉過頭來搭訕：「他以前是公務員，那副身家吃一世也吃不完。」說到這兒，黃伯慣性地說：「我擺天光墟有廿年，廿年前買樓三十五萬，現在升至三百五十萬。我有四個女，一個仔……」風雨不改，連打風也開檔，就是為了主持吹水區。睡不着的人都來這裏，中年的有，老年的也有。吹水的話題無所不有，光頭佬自豪地展示出一部舊吸塵機，鄰檔檔主的東莞婆就和街坊議論紛紛。街坊說哪有人買吸塵機？似是無無聊聊的話題說個不休。

市集和天光墟的核心價值，其實都是吹水。

後來有人坐下來閒聊幾句，不知從何說起，道顯乾脆即興創作，老本行就是寫詩，用自己最舒服的方法觀察這個世界。他邊聽邊寫，詩未必完整，但他卻喜歡內容隨情感創作。一開始大家不知從何說起，咿咿哦哦好一陣，道顯引導他們從兒時說起，例如有人說父母從小給自己強烈的約束感，現在人大了就凡事都挑戰；又有人說小時候經常見父母爭執，對婚姻沒期望等。結果，說故事的朋友們忽然滔滔不絕。他心想「可能我們都缺乏一個談自己的機會。」
#　　#　　#　　#
天漸亮了，將近八時，光頭佬豪氣地說：「好！收皮！」然後不慌不忙地收拾，最後拿着他那個用來收錢的「權杖」網球拍，推着「龍椅」回到存放區，又趕着找垃圾婆收垃圾，回家睡覺。
#　　#　　#　　#
道顯睡醒了，想到，「人工雖少，但做自己喜歡的工作實是難得。」然而想想周遭同事，不禁慨嘆工作令人對藝術麻木，最後燃燒至極點，就好像灰被風一吹，散了。
#　　#　　#　　#
散了，黃伯一早去喝早茶，天未亮時他已說：「我屋企隻乸仲瞓緊覺，快啲天光啦，就可以去飲茶，嘩！嗰啲潮州豆腐最正！潮州豆腐的做法是將鹽……」
#　　#　　#　　#
豆腐，是黃伯的盼望。
道顯摸摸兩邊褲袋說：「要將願望和人生問題隨身放在褲袋，隨時拿出來檢視自己，認清盼望。」

市集背後　搞手用力推

推廣本土手作人　A nice place to

原本是攝影工作室，每隔數月在這裏搞市集，「港式市集，就是在香港一個地方，讓香港人認識本土手作人。」負責人嬌嬌說。今年A nice place to市集第一炮，主題定為「年之初」，市集裏放了一個紙籃和願望星星紙，邀請大家寫上今年目標，「半年後我們會在這裏重聚，檢視自己目標完成了多少。」

嬌嬌

細味獨有的溫度　SUUUPER MARKET

SUUUPER MARKET有三個U，正代表三位勞苦功高的搞手，Crystie說：「我們既有廣告人，自己又做形象指導，大家工作忙碌，但卻愛逛市集。所以自己也搞一個，挑選我們欣賞及高質素的本地品牌合作。」名字「超市」的概念，就是要告訴港人，這裏應有盡有，照顧你衣食住行，何必貪方便幫襯大企業？「花茶的味道有層次讓人細味，天然護膚品全人手製作。」她形容市集，就像話劇，能夠即場感受到演員的演繹和氣氛，「這都是市集獨有的人情味和溫度，大企業沒有的。」她說。

Crystie

市集最難忘的事：中午十二時開檔，半小時後仍人影不見。「我緊張得冒汗，就像在產房前等待寶寶出世一樣。」她說。再過半小時，怎料過百人一下子就湧上來。

讓媽媽發揮力量　MAMASMART

昨天到科學園參與MAMASMART市集，我和一群小朋友在小草地上追逐，突然老遠傳來警衞呼喝聲，「不准在草地上。」有位妹妹在第一期死纏着警衞問：「為甚麼不能在草地上玩？」市集，某程度上就是對抗主流，「其中一個辦市集的原因，就是希望能善用香港不同空間。」MAMASMART負責人何曉華強調。相比台灣的市集，無論檔主和客人，人文質素更高，文化氣息更濃，以台北的Simple Market為例，不但在一個戶外的大空地上有藝術表演和工作坊，就連餐廳都售賣小農的自家農作物，「科學園這個地方未有好好善用空間。人人在海濱長廊踏單車，有誰會進來科學園的餐廳？」她說。
曉華原本是一名設計師，毅然辭去長工，了解農業，修讀Permaculture（永續栽培），學習如何規劃一個與大自然取得平衡的人生。「我認為母親，都是一個未被好好善用的資源。其實她們好厲害。」事實上，香港有很多所謂支持本土創作的市集，跟選美一樣有年齡限制「唔受媽咪玩」，除了戶外，母親是MAMASMART最關鍵的元素，所以，她既邀請媽媽們擺賣手製天然用品、辦工作坊講座等等。去年九月第一次辦市集，當時市集在香港還未形成一股風潮，誰知幾月間忽爾變成主流似的，「辦一個好市集不容易，要控制攤檔質素最難。現在不少市集只顧買賣，而非分享一種生活態度，會令人討厭，適得其反。」

何曉華

市集最難忘的事：曾經有位母親，患癌卧床時，好想每天為女兒留點滴，於是每天做一個布袋，繡上兩母女的回憶。幸好她痊癒了，決定將母女點滴在市集上跟大家分享。
想分享的態度：關注小農戶，關心生活，建立與大自然接軌的生活。
最愛市集：台灣戶外的Simple Market

天台探索生活　nomad nomad

nomad nomad市集在工廈的天台，可以說是工廈市集中氣氛最好的，它像一個城市綠洲，在周遭冷冰冰的工廠大廈群，就像只有這群人熱愛生活，自得其樂。尤其踏在天台上，感覺特別舒𣈱，搞手之一Kobe說：「我們希望能聚集一群手作人，既有婆婆手做的花生糖，又有插畫家即席揮毫，亦有不同工作坊等等。」雖然地方不大，擺滿檔攤，但客人總找到一個角落談天、寫生。

年輕搞手

市集最難忘的事：檔主稱讚我們播放的歌曲特別動聽，而且露天氣氛很好。
想分享的態度：努力探索不同的生活體驗。
最愛市集：nomad nomad（堅持愛自己）

年輕檔主　有話兒

人生即興have fun　MUM's NOT HOME

有想法的年輕人，特別多言，正因如此，他們喜愛市集。其他攤檔都有一張桌子，偏偏他們真的喜歡「地攤」，「我們喜歡這種親民的感覺。」檔主之一港傳說。然後他遞過一張綠色的小摺凳，只要你屁股一坐上去，就別想可以走人，「我們希望每一個客人知道設計背後的故事。」明明只是問這盞燈甚麼來頭，他們卻從二人的相識談起，「我們在澳洲working holiday時認識的，駿駒為人計劃周全，我卻常常即興。」港傳說。旅行的意義，就是讓他們明白生命不能夠每事計劃，「這個就是我們在市集裏想分享的生活態度，just have fun！」

教人欣賞珍惜　十一妞更衣室

去年十月成立，短短數月，參與十數次市集，老闆娘敏思：「市集賺得最多有兩萬，少則五千。」錢，還是其次，最重要是八卦，從每件衣物的來源、手工，談到旅遊，最後問人做盛行，「當你知道東西背後的故事，就會更加珍惜，不會亂花錢，每次花錢，都會欣賞它，知道設計者在哪個細節花心思，人對物就更長情，話題就不斷延伸，購物活動更加人性化。本地手作品牌當中，敏思特別欣賞島中坊研的設計，得知她在SUUUPER MARKET擺檔，一早預留彈藥。「客人在社交平台follow一個品牌，通常渴望有一面之緣，看看設計師的性格如何流露在設計中。」第一天她不在市集，也心急地問攤友：「島中坊研的設計師是怎樣的人？」在這些新興市集，無論搞市集，還是擺檔，都是時裝界潮人，「因工作關係，從前都是使錢唔抹腳，但自從參與市集，反而互相幫襯，互相欣賞。」攤友還買了島中坊研的小袋，作為她生日的驚喜。
